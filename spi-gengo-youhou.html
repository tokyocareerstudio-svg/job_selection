어구의 용법 (語句の用法)

밑줄 친 단어와 같은 의미·용법으로 사용된 문장을 선택지에서 고르는 유형. 하나의 단어가 문맥에 따라 다른 의미를 가지는 "다의어"를 이해하는 것이 핵심입니다. 특히 빈출 다의어 9개(取る·上げる·甘い·堅い 등)를 미리 정리해두면 시험장에서 속도가 올라갑니다.

테센 ★ 전용
강의
어구의 용법이란?
語句の用法 — 同じ言葉でも意味が違う

이 유형은 밑줄 친 단어가 문장에서 어떤 의미로 쓰였는지를 판단하고, 같은 의미로 쓰인 다른 문장을 고르는 문제입니다.

풀이법 3단계

① 밑줄 단어를 다른 단어로 바꿔본다. 제시문에서 밑줄 단어를 유의어로 치환. 예: "甘い考え" → "甘い = 달콤한? 아니, 생각이 느슨한 = 안이한"

② 같은 유의어로 치환 가능한 선택지를 찾는다.

③ 품사가 같은지도 확인. 동사/형용사/명사 등 품사가 다르면 용법이 다른 것.

빈출 다의어 — 이것만은 반드시

📝 대표 다의어 "取る"

① 手に取る → 손에 들다 (물리적 동작) ② 年を取る → 나이를 먹다 (변화) ③ 連絡を取る → 연락하다 (추상적 행위) ④ 資格を取る → 자격을 따다 (취득) ⑤ 休みを取る → 휴가를 얻다 (확보)

같은 "取る"인데 의미가 전혀 다릅니다. SPI에서는 "이 문장의 取る와 같은 의미의 取る는?"이라고 물어봅니다. 각 의미별로 유의어 치환이 가능한지로 판단합니다.

다의어의 핵심 = "유의어 치환"으로 같은 의미인지 확인

빈출 다의어 9개: 取る, 上げる, 打つ, 引く, 甘い, 堅い(固い/硬い), 明るい, かける, つく

이 9개 단어의 주요 의미(각 3~5개)를 정리해두면 대부분의 어구의 용법 문제에 대응 가능합니다.

⚠️ 함정 — "堅い" vs "固い" vs "硬い"

堅い = 의지가 굳건하다 (堅い決意) 固い = 단단하다, 경직되어 있다 (固い表情) 硬い = 물리적으로 딱딱하다 (硬いパン) SPI에서는 한자 표기 없이 히라가나 "かたい"로 출제하여 어떤 의미인지 문맥으로 판단하게 합니다.

문맥의 대상이 물리적 물체 → 硬い, 사람의 태도/표정 → 固い, 의지/결심 → 堅い SPI는 이 세 가지 중 "같은 かたい"를 고르라고 합니다.

히라가나만 보고 판단하면 안 됨. 반드시 문맥으로!
자주 하는 실수

실수 1: 밑줄 단어의 "사전적 대표 의미"만 떠올려서, 문맥상의 특수 의미를 놓침.

실수 2: 품사 불일치. "明るい性格(밝은 성격)"의 明るい(형용사)와 "明るくなった(밝아졌다)"의 明るく(부사적)는 같은 용법이지만, "明るみ(노출)"는 명사형으로 다른 용법.

연습 문제
어구의 용법 예제 10문제
語句の用法の練習問題10問

밑줄 친 단어와 같은 의미·용법으로 쓰인 문장을 고르세요.

Q01
取る — 물리적 동작
テーブルの上の本を取ってください。 ア 彼は今年30歳を取った。 イ 棚から皿を取った。 ウ 夏休みを取ることにした。
테이블 위의 책을 집어주세요. (물리적으로 집다) ア 그는 올해 30살이 되었다 イ 선반에서 접시를 집었다 ウ 여름 휴가를 쓰기로 했다
풀이
제시문: 取る = 물리적으로 집다 イ "선반에서 접시를 집다" → 같은 물리적 동작 ✓ ア 나이를 먹다 / ウ 휴가를 얻다 → 다른 의미
정답: イ
Q02
甘い — 판단이 느슨한
彼の計画は甘いと思う。 ア このケーキは甘い。　イ 甘い香りがする。　ウ 自分に甘い人だ。
그의 계획은 안이하다. ア 이 케이크는 달다 イ 달콤한 향기가 난다 ウ 자신에게 관대한 사람이다
풀이
제시문: 甘い = 판단이 느슨하다/안이하다 ウ "자신에게 甘い" = 자신에게 느슨하다 → 같은 의미 ✓ ア 미각의 달다 / イ 향기가 달콤 → 다른 의미
정답: ウ
Q03
上げる — 완료 보조
レポートを書き上げた。 ア 声を上げる。　イ 成績を上げる。　ウ 編み物を仕上げた。
리포트를 다 썼다(완성). ア 소리를 지르다 イ 성적을 올리다 ウ 뜨개질을 완성했다
풀이
제시문: 書き上げる = 다 쓰다(완료 보조동사) ウ 仕上げる = 완성하다 → 같은 "완료" 용법 ✓ ア/イ = 물리적으로 올리다 → 다른 의미
정답: ウ
Q04
引く — 사전을 찾다
辞書を引く。 ア 綱を引く。　イ 風邪を引く。　ウ 例を引く。
사전을 찾다. ア 줄을 당기다 イ 감기에 걸리다 ウ 예를 인용하다
풀이
辞書を引く = 사전에서 찾아보다 (참조·인용의 의미) ウ 例を引く = 예를 인용하다 → "참조"의 의미로 같음 ✓
정답: ウ
Q05
かたい — 의지가 굳건한
彼女の決意はかたい。 ア このパンはかたい。　イ かたい握手を交わした。　ウ 合格はかたいだろう。
그녀의 결의는 굳건하다. ア 이 빵은 딱딱하다 イ 굳은 악수를 나눴다 ウ 합격은 확실할 것이다
풀이
제시문: かたい決意 = 굳건한 결의 (堅い) イ かたい握手 = 굳건한 악수 → 의지·유대의 "굳건함" ✓ ア 물리적 딱딱함(硬い) / ウ 확실하다(堅い지만 "가능성" 의미)
정답: イ
Q06
打つ — 행동을 취하다
対策を打つ。 ア 釘を打つ。　イ 電報を打つ。　ウ 手を打つ。
대책을 세우다. ア 못을 박다 イ 전보를 치다 ウ 손을 치다(합의하다)
풀이
対策を打つ = 대책을 강구하다(추상적 행동) ウ 手を打つ = 합의를 보다/손을 쓰다 → 추상적 행동 ✓ ア 물리적 타격 / イ 전보를 보내다(통신 행위)
정답: ウ
Q07
明るい — 지식이 풍부한
彼はIT分野に明るい。 ア 明るい部屋。　イ 明るい性格。　ウ 政治に明るい人。
그는 IT 분야에 밝다(정통하다). ア 밝은 방 イ 밝은 성격 ウ 정치에 밝은 사람
풀이
IT分野に明るい = 잘 알고 있다(정통) ウ 政治に明るい = 정치에 정통 → 같은 의미 ✓
정답: ウ
Q08
かける — 시간·비용을 들이다
この作品には3年をかけた。 ア 壁に絵をかける。　イ 迷惑をかける。　ウ 時間をかけて作る。
이 작품에 3년을 들였다. ア 벽에 그림을 걸다 イ 폐를 끼치다 ウ 시간을 들여 만들다
풀이
3年をかけた = 시간을 투입하다 ウ 時間をかけて = 시간을 들여 → 같은 의미 ✓
정답: ウ
Q09
つく — 상태가 되다
自信がついた。 ア 嘘をつく。　イ 力がついた。　ウ 机につく。
자신감이 붙었다. ア 거짓말을 하다 イ 힘이 붙었다 ウ 책상에 앉다
풀이
自信がつく = 자신감이 생기다(상태 획득) イ 力がつく = 힘이 생기다 → 같은 구조·의미 ✓
정답: イ
Q10
上げる — 물리적으로 올리다
荷物を棚に上げた。 ア 成果を上げる。　イ 天ぷらを上げる。　ウ 旗を上げる。
짐을 선반에 올렸다. ア 성과를 올리다 イ 튀김을 올리다(건지다) ウ 깃발을 올리다
풀이
荷物を棚に上げる = 물리적으로 위로 올리다 ウ 旗を上げる = 깃발을 물리적으로 올리다 ✓ ア 추상적 성과 / イ 기름에서 건지다(揚げる)
정답: ウ
