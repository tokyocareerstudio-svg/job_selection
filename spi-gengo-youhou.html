어구의 용법 (語句の用法)

밑줄 친 단어와 "같은 의미로 사용된 문장"을 선택지에서 고르는 유형. 같은 단어라도 문맥에 따라 의미가 달라지는 다의어(多義語)의 용법을 구분하는 능력이 시험됩니다.

테센 ★ 전용
강의
어구의 용법이란?
語句の用法 — 同じ単語の違う意味を見分ける

이 유형은 SPI 언어에서 난이도 "중상"에 해당합니다. 문제에서 밑줄 친 단어가 어떤 의미로 사용되었는지 파악하고, 선택지에서 같은 의미로 쓰인 문장을 고르는 형식입니다. 테스트센터 전용.

일본어의 동사·형용사는 한 단어가 여러 의미를 갖는 경우가 많습니다. 예를 들어 "かける"만 해도 "전화를 걸다", "안경을 쓰다", "시간이 걸리다", "자물쇠를 잠그다", "다리를 꼬다" 등 10가지 이상의 의미가 있습니다.

풀이 핵심 3가지

① 조사(助詞)에 주목하라. に·を·が 등의 조사가 달라지면 동사의 의미가 달라집니다. "電話をかける"(전화를 걸다)와 "鍵をかける"(자물쇠를 잠그다)는 같은 をかける이지만 의미가 다름.

② 물리적 vs 비유적을 구분하라. "手を広げる"가 물리적으로 손을 펼치는 것인지, 비유적으로 사업을 넓히는 것인지. 문제의 밑줄 단어가 어떤 쪽인지 먼저 판단.

③ 문장 전체 문맥으로 판단하라. 단어만 보지 말고 문장 전체의 상황을 파악해야 합니다.

출제 형태 상세

📝 미니 예제 — かける의 용법

彼女に電話をかける ア　壁に絵をかける　　イ　友人に声をかける ウ　鍵をかける　　エ　時間がかかる

풀이: "전화를 걸다" → "~에게 ~을 전달/발신하다"의 의미. 각 선택지를 분석: ア "벽에 그림을 걸다" = 물리적으로 매달다 ✗ イ "친구에게 말을 걸다" = ~에게 ~을 전달하다 ✓ (같은 용법!) ウ "자물쇠를 잠그다" = 고정시키다 ✗ エ "시간이 걸리다" = 소요되다 (자동사, かかる) ✗

정답: イ 友人に声をかける

빈출 함정 패턴

SPI에서 자주 나오는 다의어 함정 패턴을 미리 알아두세요.

⚠️ 함정 — 물리적 vs 비유적

あの人は口がかたい (그 사람은 입이 무겁다) ア　このパンはかたい (이 빵은 딱딱하다) イ　彼は頭がかたい (그는 머리가 딱딱하다) ウ　彼女は意志がかたい (그녀는 의지가 굳다)

함정: "입이 무겁다" = 비밀을 잘 지킨다 → 성격·기질에 대한 비유적 표현. ア 물리적 경도(硬い) ✗ イ 융통성 없음(비유이긴 하지만 "고집스럽다"의 뉘앙스) △ ウ "의지가 굳다" = 성격·기질이 확고함 → 같은 비유적 용법 ✓

정답: ウ 意志がかたい
📝 미니 예제 — とる의 용법

大学で資格をとった (대학에서 자격을 땄다) ア　テーブルの本をとる (테이블 위 책을 잡다) イ　写真をとる (사진을 찍다) ウ　運転免許をとる (운전면허를 따다)

풀이: "자격을 따다" = 획득하다(取得する). ア 물리적으로 집다 / イ 촬영하다(撮る) / ウ "면허를 따다" = 획득하다 ✓

정답: ウ 運転免許をとる
효율적 학습법

빈출 다의어를 단어별로 정리해서 외우는 것이 가장 효율적입니다. 아래 "빈출 다의어" 섹션에서 かける·とる·あげる·つく 등 자주 출제되는 단어별로 용법을 정리했습니다. 각 단어의 용법 차이를 "조사+문맥"으로 구분하는 연습을 반복하세요.

한국인 수험생 주의

한국어 감각에 의존하지 마세요. 한국어 "걸다"는 "전화를 걸다"와 "그림을 걸다"가 같은 단어이지만, 일본어에서는 문맥에 따라 완전히 다른 용법으로 분류됩니다. 반드시 일본어 문맥 안에서 의미를 판단하는 습관을 들이세요.

밑줄 친 단어와 같은 의미로 사용된 문장을 고르세요. 조사와 문맥에 주목!

Q01
かける
下線の語が最も近い意味で使われているものを選びなさい。 彼女に電話をかける ア　壁に絵をかける イ　友人に声をかける ウ　鍵をかける エ　時間がかかる
"그녀에게 전화를 걸다"와 같은 의미는? ア 벽에 그림을 걸다 イ 친구에게 말을 걸다 ウ 자물쇠를 잠그다 エ 시간이 걸리다
풀이
"전화를 걸다" = ~에게 ~을 전달하다. イ "말을 걸다" = ~에게 ~을 전달 → 같은 용법 ✓ ア 물리적으로 걸다 / ウ 고정하다 / エ 소요되다
정답: イ 友人に声をかける
Q02
とる
彼は大学で資格をとった ア　テーブルの上の本をとる イ　旅行の写真をとる ウ　運転免許をとる エ　魚をとる
"대학에서 자격을 땄다"와 같은 의미는? ア 테이블 위 책을 잡다 イ 여행 사진을 찍다 ウ 운전면허를 따다 エ 물고기를 잡다
풀이
"자격을 따다" = 획득하다. ウ "면허를 따다" = 획득하다 → 같은 용법 ✓ ア 물리적으로 집다 / イ 촬영하다 / エ 포획하다
정답: ウ 運転免許をとる
Q03
あげる
友達に誕生日プレゼントをあげた ア　天ぷらを油であげる イ　大きな声をあげる ウ　弟に本をあげる エ　成果をあげる
"친구에게 생일 선물을 줬다"와 같은 의미는? ア 튀김을 기름에 튀기다 イ 큰 소리를 지르다 ウ 동생에게 책을 주다 エ 성과를 올리다
풀이
"선물을 주다" = 타인에게 물건을 제공. ウ "책을 주다" = 같은 제공의 용법 ✓ ア 조리법 / イ 발성 / エ 달성
정답: ウ 弟に本をあげる
Q04
つく
新しい仕事につく ア　テーブルに傷がつく イ　嘘をつく ウ　教壇につく エ　電気がつく
"새 직장에 취임하다"와 같은 의미는? ア 테이블에 상처가 나다 イ 거짓말을 하다 ウ 교단에 서다 エ 전기가 켜지다
풀이
"직에 취임하다" = 지위·역할에 서다. ウ "교단에 서다" = 위치·역할에 서다 → 같은 용법 ✓ ア 흠집 / イ 거짓말 / エ 점등
정답: ウ 教壇につく
Q05
かたい
あの人は口がかたい ア　このパンはかたい イ　彼は頭がかたい ウ　約束はかたい エ　彼女は意志がかたい
"그 사람은 입이 무겁다"와 같은 의미는? ア 이 빵은 딱딱하다 イ 그는 머리가 딱딱하다 ウ 약속은 확실하다 エ 그녀는 의지가 굳다
풀이
"입이 무겁다" = 비밀을 잘 지킨다 (비유적·성격). エ "의지가 굳다" = 확고부동 (비유적·성격) → 같은 용법 ✓ ア 물리적 경도 / イ 융통성 없음 / ウ 확실함
정답: エ 意志がかたい
Q06
はかる
事業の拡大をはかる ア　体温をはかる イ　利益をはかる ウ　時間をはかる エ　距離をはかる
"사업 확대를 도모하다"와 같은 의미는? ア 체온을 재다 イ 이익을 도모하다 ウ 시간을 재다 エ 거리를 재다
풀이
"확대를 도모하다" = 계획·추진하다. イ "이익을 도모하다" = 계획·추진 → 같은 용법 ✓ ア/ウ/エ 측정하다 (계る·量る)
정답: イ 利益をはかる
Q07
おさめる
国をおさめる ア　税金をおさめる イ　成果をおさめる ウ　箱におさめる エ　争いをおさめる
"나라를 다스리다"와 같은 의미는? ア 세금을 납부하다 イ 성과를 거두다 ウ 상자에 넣다 エ 다툼을 가라앉히다
풀이
"나라를 다스리다" = 통치·관리하다 (治める). エ "다툼을 가라앉히다" = 수습하다 (治める) → 같은 한자 ✓ ア 納める / イ 収める / ウ 収める
정답: エ 争いをおさめる
Q08
あつい
彼は人情にあつい ア　今日はあつい日だ イ　この本はあつい ウ　あつい友情で結ばれている エ　お湯があつい
"그는 인정이 두텁다"와 같은 의미는? ア 오늘은 더운 날이다 イ 이 책은 두껍다 ウ 두터운 우정으로 맺어져 있다 エ 물이 뜨겁다
풀이
"인정에 두텁다" = 마음이 깊고 따뜻하다 (篤い). ウ "두터운 우정" = 깊은 관계 (篤い) → 같은 용법 ✓ ア 暑い / イ 厚い(물리) / エ 熱い
정답: ウ あつい友情
Q09
きく
融通がきく ア　ラジオをきく イ　薬がきく ウ　鼻がきく エ　道をきく
"융통이 되다"와 같은 의미는? ア 라디오를 듣다 イ 약이 듣다 ウ 코가 예리하다 エ 길을 묻다
풀이
"융통이 きく" = 기능이 발휘되다. ウ "코가 きく" = 감각이 발휘되다 → 같은 용법 ✓ ア 聞く(청취) / イ 効く(효과) / エ 聞く(질문)
정답: ウ 鼻がきく
Q10
のぼる
被害額は1億円にのぼる ア　山にのぼる イ　話題にのぼる ウ　出世の階段をのぼる エ　総額が100万にのぼる
"피해액이 1억 엔에 달하다"와 같은 의미는? ア 산에 오르다 イ 화제에 오르다 ウ 출세의 계단을 오르다 エ 총액이 100만에 달하다
풀이
"~에 달하다" = 수량이 그 정도에 이르다. エ "총액이 100만에 달하다" = 같은 수량 도달 용법 ✓ ア 물리적 등산 / イ 화제 / ウ 비유적 상승
정답: エ 総額が100万にのぼる
