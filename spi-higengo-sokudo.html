속도·일·농도 (速度·仕事·濃度)

중학교 수학의 "문장제"가 그대로 나오는 유형. 속도(거리=속도×시간), 일산(전체=1), 농도(용질/용액), 연령산(나이 차 불변), 통과산(열차 길이 포함) 등을 다룹니다. 공식을 알면 어렵지 않지만, 단위 변환과 조건 읽기 실수가 잦은 유형입니다.

속도·일·농도 — 문장제의 정석
速度·仕事·濃度 — 文章題の定石

이 유형에서 가장 중요한 것은 공식 자체를 아는 것과 단위를 맞추는 것입니다. "시속 4km인데 시간이 30분"이면, 30분을 0.5시간으로 변환해야 합니다. 이 변환 실수가 이 유형의 실점 원인 1위입니다.

이 유형의 대원칙

① 단위를 반드시 맞춰라. km/h와 분, m/분과 시간 등 단위가 섞여 나옴. 계산 전에 단위 통일!

② "전체를 1로 놓기"에 익숙해져라. 일산·농도산의 핵심 테크닉.

③ 그림을 그려라. 속도 문제는 상황도, 농도 문제는 소금물 그림을 그리면 실수가 줄어듦.

속도산 (速度算) — 거리·속도·시간의 삼각형

속도 문제의 모든 것은 아래 하나의 관계에서 나옵니다. 일본에서는 "は·じ·き"(速さ·時間·距離)라고 외웁니다.

"は·じ·き" 삼각형
거리 (き)
속도 (は)
시간 (じ)

거리 = 속도 × 시간 / 속도 = 거리 ÷ 시간 / 시간 = 거리 ÷ 속도

속도 기본
거리 = 속도 × 시간
시속 4km로 30분(=0.5시간) → 거리 = 4 × 0.5 = 2km
📝 미니 예제

시속 60km로 2시간 30분 달리면 거리는?

2시간 30분 = 2.5시간 ← 단위 변환 필수! 거리 = 60 × 2.5 = 150km

정답: 150km

상대 속도 — 따라잡기와 마주오기

같은 방향 (따라잡기/追いかけ)
상대속도 = 빠른 속도 - 느린 속도
시속 80km가 시속 60km를 뒤쫓음 → 상대속도 = 20km/h
반대 방향 (마주오기/出会い)
상대속도 = 속도A + 속도B
시속 60km와 시속 40km가 마주 달림 → 상대속도 = 100km/h
📝 미니 예제

A(시속 80km)가 B(시속 60km)보다 10km 뒤에서 출발. A가 B를 따라잡는 데 걸리는 시간은?

상대속도 = 80 - 60 = 20km/h 따라잡아야 할 거리 = 10km 시간 = 10 ÷ 20 = 0.5시간 (30분)

정답: 30분

통과산 (通過算) — 웹테스팅 전용

웹테스팅에서만 출제. 열차가 다리나 터널을 "완전히 통과"하려면 열차의 길이도 거리에 포함해야 합니다.

통과산
이동 거리 = 열차 길이 + 대상 길이
열차 100m가 다리 200m를 통과 → 이동 거리 = 300m
📝 미니 예제

길이 150m의 열차가 초속 25m로 달린다. 길이 350m의 다리를 완전히 통과하는 데 몇 초?

이동 거리 = 150(열차) + 350(다리) = 500m 시간 = 500 ÷ 25 = 20초

정답: 20초

일산 (仕事算) — "전체를 1로 놓기"

일산의 핵심 테크닉은 전체 일의 양을 1로 놓는 것입니다. A가 6일 걸리면 1일 작업량 = 1/6, B가 12일이면 1/12. 함께 하면 합산.

일산 기본
1일 작업량 = 1 / 소요일수
A가 6일 → 1일 = 1/6. B가 12일 → 1일 = 1/12. 합산 = 1/6+1/12 = 1/4 → 4일
📝 미니 예제

수조에 물을 채우는 데 A호스로 6시간, B호스로 3시간 걸린다. 양쪽 동시에 열면?

A: 1시간 = 1/6 / B: 1시간 = 1/3 합산 = 1/6 + 1/3 = 1/6 + 2/6 = 3/6 = 1/2 소요 = 1 ÷ (1/2) = 2시간

정답: 2시간

농도산 (濃度算) — 소금물 문제

"소금물 문제"로 유명한 유형. 용질(소금), 용매(물), 용액(소금물)의 관계를 이해하면 됩니다.

농도 기본
농도(%) = 용질 / 용액 × 100
소금 10g, 소금물 200g → 농도 = 10/200 × 100 = 5%
농도 혼합
(농도A×양A + 농도B×양B) / (양A + 양B)
5% 200g + 10% 300g = (10+30)/500 = 8%
📝 미니 예제

8% 소금물 300g에서 물을 100g 증발시키면 농도는?

소금(용질) = 300 × 0.08 = 24g ← 소금은 변하지 않음! 증발 후 용액 = 300 - 100 = 200g 농도 = 24/200 × 100 = 12%

정답: 12%

연령산 (年齢算) — 웹테스팅 전용

웹테스팅에서만 출제. 핵심 원리: 두 사람의 나이 차이는 시간이 지나도 절대 변하지 않는다.

연령산 핵심
나이 차이 = 항상 일정
아버지 40, 아들 12 → 차이 28. 10년 후에도 50과 22, 차이 28.
📝 미니 예제

현재 형 18세, 동생 12세. 형의 나이가 동생의 2배였던 것은 몇 년 전?

X년 전: 형 18-X, 동생 12-X 18-X = 2(12-X) → 18-X = 24-2X → X = 6 검증: 6년 전 형 12, 동생 6 → 12/6 = 2배 ✓

정답: 6년 전
자주 하는 실수 TOP 3

실수 1: 단위 변환 누락. "시속 4km, 30분"에서 0.5시간으로 변환 안 함.

실수 2: 농도에서 물을 증발시켰는데 소금까지 줄어든다고 계산. 용질(소금)은 변하지 않음!

실수 3: 통과산에서 열차 길이를 빼먹음. "완전히 통과" = 열차 길이 + 대상 길이.

Q01 — 속도 기본
자전거 속도 구하기
Aさんは家から学校まで時速4kmで歩くと30分かかる。自転車で行くと10分で着く。自転車の速度は時速何kmか。
A씨는 집에서 학교까지 시속 4km로 걸으면 30분. 자전거로 10분. 자전거 속도는?
풀이
거리 = 4 × 0.5(시간) = 2km 자전거 속도 = 2 ÷ (10/60) = 2 ÷ (1/6) = 12km/h
정답: 시속 12km
Q02 — 마주오기
출발 후 몇 분에 만남?
AとBが10km離れた地点からお互いに向かって同時に出発した。Aは時速6km、Bは時速4kmで歩く。2人が出会うのは出発から何分後か。
A와 B가 10km 떨어진 지점에서 서로를 향해 동시 출발. A 시속 6km, B 시속 4km. 만나는 건 출발 몇 분 후?
풀이
상대속도 = 6+4 = 10km/h 시간 = 10km ÷ 10km/h = 1시간 = 60분
정답: 60분 후
Q03 — 일산
함께 일하면 며칠?
ある仕事をAさん1人ですると10日、Bさん1人ですると15日かかる。AとBが一緒にすると何日で終わるか。
A 혼자 10일, B 혼자 15일. 함께 하면 며칠?
풀이
A: 1/10 + B: 1/15 = 3/30+2/30 = 5/30 = 1/6 소요: 6일
정답: 6일
Q04 — 농도 혼합
소금물 섞기
5%の食塩水200gと12%の食塩水300gを混ぜると、濃度は何%になるか。
5% 소금물 200g과 12% 소금물 300g을 섞으면 농도는?
풀이
소금: 200×0.05 + 300×0.12 = 10+36 = 46g 용액: 200+300 = 500g 농도: 46/500 × 100 = 9.2%
정답: 9.2%
Q05 — 연령산 웹테
나이의 3배
現在、父は40歳、息子は12歳である。父の年齢が息子の年齢の3倍になるのは何年後か。
아버지 40세, 아들 12세. 아버지가 아들의 3배가 되는 건 몇 년 후?
풀이
X년 후: 40+X = 3(12+X) 40+X = 36+3X → 4 = 2X → X = 2 검증: 42/14 = 3 ✓
정답: 2년 후
Q06 — 통과산 웹테
열차가 다리를 통과
長さ200mの列車が秒速20mで走っている。長さ300mの橋を完全に渡り終えるのに何秒かかるか。
길이 200m 열차가 초속 20m로 달림. 길이 300m 다리를 완전히 건너는 데 몇 초?
풀이
이동 거리 = 200(열차)+300(다리) = 500m 시간 = 500 ÷ 20 = 25초
정답: 25초
Q07 — 평균 속도
왕복 평균 속도
AさんはA地点からB地点まで時速6kmで行き、帰りは時速4kmで戻った。A地点からB地点までの距離は12kmである。往復の平均速度は時速何kmか。
A에서 B까지 시속 6km로 가고, 돌아올 때 시속 4km. 거리 12km. 왕복 평균 속도는?
풀이
총거리 = 12×2 = 24km 갈 때: 12÷6 = 2시간 / 올 때: 12÷4 = 3시간 / 총시간 = 5시간 평균속도 = 24÷5 = 4.8km/h ※ 주의: (6+4)÷2 = 5가 아님! 반드시 총거리/총시간으로 계산.
정답: 시속 4.8km
Q08 — 농도 (물 추가)
물을 더하면?
10%の食塩水400gに水を100g加えると、濃度は何%になるか。
10% 소금물 400g에 물 100g을 추가하면 농도는?
풀이
소금 = 400×0.10 = 40g (변하지 않음) 용액 = 400+100 = 500g 농도 = 40/500 × 100 = 8%
정답: 8%
Q09 — 일산 응용
도중에 합류
ある仕事をAさんが1人で5日間行った後、Bさんが加わって一緒に残りを仕上げた。Aは全体を12日、Bは全体を6日で終える力がある。Bが加わってから何日で終わったか。
A가 혼자 5일 한 후 B가 합류. A는 전체 12일, B는 전체 6일에 끝내는 능력. B 합류 후 며칠 걸렸나?
풀이
A의 5일 작업량: 5 × (1/12) = 5/12 남은 작업량: 1 - 5/12 = 7/12 합산 작업량/일: 1/12 + 1/6 = 1/12 + 2/12 = 3/12 = 1/4 남은 일수: (7/12) ÷ (1/4) = (7/12) × 4 = 28/12 = 7/3 ≈ 2.33일
정답: 7/3일 (약 2일과 1/3일, 즉 2일 8시간)
Q10 — 따라잡기 응용
형과 동생
弟が家を出発してから8分後に、兄が追いかけた。弟は分速60m、兄は分速100mで歩く。兄が弟に追いつくのは、兄が出発してから何分後か。
동생이 출발한 지 8분 후 형이 뒤쫓음. 동생 분속 60m, 형 분속 100m. 형이 따라잡는 건 형 출발 몇 분 후?
풀이
동생 선행 거리: 60 × 8 = 480m 상대속도: 100 - 60 = 40m/분 따라잡기 시간: 480 ÷ 40 = 12분
정답: 12분 후
