SPI 비언어와의 핵심 차이

SPI 비언어는 순열·조합, 확률, 추론 등 다양한 유형이 섞여 출제되지만, 玉手箱 계수는 1형식만 끝까지 반복 출제됩니다. 또한 웹 시험(자택 수험)에서는 전자계산기 사용이 전제되어 있어, 계산 자체의 난도보다 제한 시간 내 정확한 처리 속도가 승패를 가릅니다.

또 하나의 차이는 오류율(誤謬率)이 측정되지 않는다는 것입니다. 틀려도 감점이 없으므로, 모르는 문제는 빠르게 찍고 넘기는 것이 정석입니다.

玉手箱 계수의 가장 큰 특징은 과목당 1형식만 끝까지 반복 출제된다는 것입니다. 최초 문제가 사칙역산이면 마지막까지 사칙역산만 나옵니다. 어떤 형식이 나올지 사전에 알 수 없으므로 3형식 모두 대비해야 하지만, 문제 수와 제한 시간으로 형식을 판별할 수 있습니다.

C-GAB(테스트센터) 주의

C-GAB는 테스트센터에서 치르는 玉手箱 파생 시험입니다. 전자계산기 사용 불가이므로, 암산·필산·개산(概算) 능력이 필요합니다. 웹 시험에서 계산기에 의존하는 습관을 들이면 C-GAB에서 시간 부족에 빠질 수 있으므로, 가능하면 암산 연습도 병행하세요.

3형식 공통 원칙

• 전자계산기를 미리 준비 — 웹 시험에서는 PC 내장 계산기 또는 실물 계산기 사용 가능. 시험 시작 전 계산기를 열어두기
• 오류율 미측정 — 모르면 찍고 넘기기. 빈칸으로 두는 것보다 아무 값이나 선택하는 것이 유리
• 되돌아갈 수 없음 — 다음 문제로 넘기면 이전 문제 재열람 불가
• 단위에 주의 — 만(万), 억(億), %(퍼센트), 전년대비 등 단위 혼동이 가장 흔한 실수

학습 전략
効率的な対策ロードマップ
추천 학습 순서

STEP 1. 사칙역산부터 마스터
가장 빈출이며, 1문당 10초라는 극한의 속도가 요구됨. 이항(移項) 기법을 완벽히 익히고, 계산기 조작을 손에 익히는 것이 최우선.

STEP 2. 도표읽기
비율·변화율·전년대비 공식을 암기하고, 도표에서 필요한 수치만 빠르게 추출하는 연습. 불필요한 데이터를 무시하는 능력이 핵심.

STEP 3. 표빈칸추측
표의 법칙성을 발견하는 사고력이 필요. 비례·반비례·합계 활용 등 패턴을 많이 접해두면 유리.

STEP 4. 실전 시간 관리 연습
반드시 타이머를 켜고 실전과 동일한 제한 시간에서 연습. 특히 사칙역산 50문/9분은 체감해보지 않으면 속도감을 잡기 어려움.

전자계산기 활용 팁

• Windows 계산기 단축키 — Win+R → calc 입력. 「표준」 모드로 충분
• Mac 계산기 — Spotlight(Cmd+Space) → "계산기" 검색
• 실물 계산기 추천 — 화면을 보면서 손가락으로 치는 것이 PC 계산기보다 빠른 경우가 많음
• M+ / MR 기능 활용 — 중간 결과를 메모리에 저장하면 복잡한 계산에서 시간 단축
• % 키 활용 — 도표읽기에서 비율 계산 시 유용 (예: 150 x 23% = 150 × 23 %)

C-GAB 대비 포인트 (암산 훈련)

• 개산(概算) 습관 — 선택지가 5개라면, 대략적인 값으로 2-3개를 먼저 소거 가능
• 25% = 1/4, 33% = 1/3, 20% = 1/5 — 분수 변환으로 암산 속도 향상
• 끝자리 계산 — 선택지의 끝자리만 다르면, 끝자리 계산으로 정답 특정 가능
