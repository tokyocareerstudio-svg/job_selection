웹테 전용 유형의 특징

① 답을 직접 입력. 5지선다가 아니라 숫자를 직접 타이핑. 찍기 불가.

② 전자계산기 사용 가능. 복잡한 계산은 계산기로 처리. 그 대신 문제 난이도가 높음.

③ "식을 세우는 힘"이 핵심. 계산 자체보다 문제를 식으로 변환하는 능력이 시험됨.

정수의 추리 (整数の推理) — 수열 규칙 찾기

나열된 숫자의 규칙을 발견하여 빈칸을 채우는 유형. 웹테에서 가장 자주 나옵니다.

접근법: ① 인접한 수의 차이(계차)를 구해본다 ② 차이가 일정하지 않으면 "차이의 차이"도 확인 ③ 차이가 아니라 비율(×2, ×3 등)인지 확인 ④ ×2+1, ×3-2 같은 복합 규칙도 시도

예시: 2, 5, 11, 23, ?, 95

23×2+1 = 47 / 47×2+1 = 95 ✓ → 답: 47

⚠️ 함정 — 계차가 아니라 비율인 경우

3, 6, 12, 24, ? 차이: 3, 6, 12 → 일정하지 않음. 비율을 보면?

각 항이 전 항의 ×2: 3→6→12→24→48 차이만 보면 혼란스럽지만, 비율로 보면 단순한 등비수열.

계차가 안 되면 비율(×, ÷)을 반드시 확인!

방정식 (方程式) — 문장을 식으로

문장의 조건을 x, y 등의 변수로 놓고 방정식을 세워 푸는 유형입니다.

1차 방정식
ax + b = c → x = (c - b) / a
"어떤 수의 3배에 5를 더하면 20" → 3x+5=20 → x=5
📝 미니 예제

어떤 수에 7을 더한 뒤 3배 하면 45가 된다. 어떤 수는?

x로 놓으면: 3(x+7) = 45 3x+21 = 45 → 3x = 24 → x = 8

정답: 8

연립방정식 — 미지수 2개 → 식 2개

미지수가 2개면 식도 2개 필요합니다. 가감법 또는 대입법으로 풉니다.

연립방정식 (가감법)
식1 × n + 식2 × m → 한 변수 소거
"합"과 "차"가 주어지는 패턴이 가장 빈출
📝 미니 예제

사과 + 귤 = 10개, 사과 - 귤 = 2개. 사과는 몇 개?

x+y=10, x-y=2 더하면: 2x=12 → x=6 → y=4

정답: 사과 6개

배수·약수 (倍数) — 판별법 암기

특정 수의 배수인지 빠르게 판별하는 방법을 알면 시간을 크게 절약합니다.

배수 판별법 정리

2의 배수: 끝자리가 짝수 (0,2,4,6,8)
3의 배수: 각 자릿수의 합이 3의 배수 (예: 123 → 1+2+3=6 → 3의 배수 ✓)
4의 배수: 끝 두 자리가 4의 배수 (예: 312 → 12÷4=3 → ✓)
5의 배수: 끝자리가 0 또는 5
9의 배수: 각 자릿수의 합이 9의 배수 (예: 729 → 7+2+9=18 → ✓)
자주 하는 실수

실수 1: 정수 추리에서 한 가지 규칙만 시도하고 포기. 계차→비율→복합 순서로 최소 3가지는 시도할 것.

실수 2: 연립방정식에서 식을 잘못 세움. 문제를 읽고 "합이 ~" "차가 ~"를 정확히 식으로 변환.

실수 3: 최소공배수와 최대공약수를 혼동. 최소공배수(LCM) = 두 수를 공통으로 나눌 수 있는 가장 작은 수. 최대공약수(GCD) = 공통 약수 중 가장 큰 수.

Q01 — 정수의 추리
수열 빈칸 채우기 (×2+1)
次の数列の□に入る数字を答えなさい。 2, 5, 11, 23, □, 95
다음 수열의 □에 들어갈 숫자는? 2, 5, 11, 23, □, 95
풀이
규칙: ×2+1 2→5→11→23→47→95 ✓
정답: 47
Q02 — 정수의 추리
수열 빈칸 (계차 등비)
次の数列の□に入る数字を答えなさい。 1, 2, 4, 8, 16, □
다음 수열의 □는? 1, 2, 4, 8, 16, □
풀이
각 항 ×2: 1→2→4→8→16→32
정답: 32
Q03 — 정수의 추리
수열 빈칸 (계차 증가)
次の数列の□に入る数字を答えなさい。 3, 5, 9, 15, 23, □
다음 수열의 □는? 3, 5, 9, 15, 23, □
풀이
계차: 2, 4, 6, 8, ... → 계차가 +2씩 증가 다음 계차 = 10 23 + 10 = 33
정답: 33
Q04 — 연립방정식
사과와 귤
りんご3個とみかん5個で1,100円、りんご2個とみかん3個で700円。りんご1個の値段は？
사과 3+귤 5=1100엔, 사과 2+귤 3=700엔. 사과 1개는?
풀이
3x+5y=1100...(1) 2x+3y=700...(2) (1)×2: 6x+10y=2200 (2)×3: 6x+9y=2100 빼기: y=100 → x=(700-300)/2=200
정답: 200엔
Q05 — 배수
3과 5의 공배수
1から100までの整数のうち、3の倍数であり、かつ5の倍数でもある数はいくつあるか。
1~100 정수 중 3의 배수이면서 5의 배수인 수는 몇 개?
풀이
최소공배수 = 15 100÷15 = 6.67 → 6개 (15,30,45,60,75,90)
정답: 6개
Q06 — 방정식
문장 → 식
ある数の4倍から7を引くと21になる。ある数はいくつか。
어떤 수의 4배에서 7을 빼면 21이 된다. 어떤 수는?
풀이
4x - 7 = 21 → 4x = 28 → x = 7
정답: 7
Q07 — 연립방정식 응용
어른과 어린이
大人と子供合わせて30人が映画を見に行った。入場料は大人800円、子供500円で、合計は18,000円だった。大人は何人か。
어른+어린이 합계 30명. 어른 800엔, 어린이 500엔, 합계 18,000엔. 어른은 몇 명?
풀이
x+y=30, 800x+500y=18000 y=30-x 대입: 800x+500(30-x)=18000 800x+15000-500x=18000 → 300x=3000 → x=10
정답: 어른 10명
Q08 — 정수 추리 (교대 패턴)
두 규칙이 교대
次の数列の□に入る数字を答えなさい。 1, 3, 2, 6, 4, 12, □
수열의 □는? 1, 3, 2, 6, 4, 12, □
풀이
홀수번째: 1, 2, 4 → ×2 짝수번째: 3, 6, 12 → ×2 7번째(홀수) → 4×2 = 8
정답: 8
Q09 — 배수 응용
4의 배수 판별
3桁の整数で、各桁の数の和が12であり、かつ4の倍数であるものを1つ答えなさい。
3자리 정수 중 각 자릿수 합이 12이고 4의 배수인 것을 1개 답하시오.
풀이
자릿수 합=12인 수: 480(4+8+0=12), 444(4+4+4=12), 372(3+7+2=12)... 4의 배수 확인(끝 두 자리÷4): 480 → 80÷4=20 ✓ 444 → 44÷4=11 ✓ 372 → 72÷4=18 ✓ 어느 것이든 정답.
정답: 480, 444, 372 등 (복수 정답 가능)
Q10 — 방정식 응용
나이 문제 (방정식으로)
現在、兄は妹より6歳年上である。5年後に兄の年齢が妹の年齢の2倍になるとき、現在の妹の年齢はいくつか。
형은 여동생보다 6살 위. 5년 후 형의 나이가 여동생의 2배가 될 때, 현재 여동생의 나이는?
풀이
여동생 = x, 형 = x+6 5년 후: (x+6+5) = 2(x+5) x+11 = 2x+10 1 = x → 여동생 = 1세 검증: 5년 후 형 12세, 여동생 6세 → 12/6=2 ✓
정답: 1세
