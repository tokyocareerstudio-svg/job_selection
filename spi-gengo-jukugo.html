숙어의 구성 (熟語の成り立ち)

이자숙어가 어떤 구조로 만들어졌는지 5가지 패턴을 판별하는 유형. 웹테스팅(WEB) 전용. "문장으로 바꿔보기" 기법으로 풀 수 있습니다.

웹테스팅 ★ 전용
강의
숙어의 구성이란?
熟語の成り立ち — 漢字を分解して文にしてみる

이 유형은 웹테스팅(WEB) 전용이며 테스트센터에서는 미출제입니다. 이자숙어의 한자 2글자가 어떤 관계로 결합되어 있는지를 5가지 패턴 중에서 판별하는 문제입니다.

실전에서는 A~D 4개 선택지 + "어디에도 해당하지 않음(E)"으로 출제됩니다. 풀이법은 "한자를 분해해서 짧은 문장으로 바꿔보기" 하나로 모든 문제를 풀 수 있습니다.

5대 패턴 상세 해설

A. 유의어 결합: 비슷한 뜻의 한자 2개를 합친 것. 두 글자를 따로 읽었을 때 뜻이 비슷하면 이 패턴.
　例: 岩石(바위+돌) → "岩도 石도 같은 뜻" → 유의어

B. 대의어 결합: 반대 뜻의 한자 2개를 합친 것.
　例: 高低(높음+낮음) → "高와 低는 반대" → 대의어

C. 주술 관계: 앞 글자가 주어, 뒷 글자가 서술어.
　例: 日没 → "日が没する(해가 진다)" → 주어+서술어

D. 동사+목적어: 뒷 글자가 앞 글자 동사의 목적어.
　例: 読書 → "書を読む(책을 읽다)" → 동사+목적어

E. 수식 관계: 앞 글자가 뒷 글자를 꾸며줌(형용사+명사).
　例: 強風 → "強い風(강한 바람)" → 수식

판별법 — "문장으로 바꿔보기"

숙어의 한자를 분해해서 짧은 일본어 문장으로 바꿔보면 관계가 드러납니다.

📝 미니 예제 — 동사+목적어

「読書」の成り立ち方として当てはまるものを選びなさい。 A 似た意味を持つ漢字を重ねる B 反対の意味を持つ漢字を重ねる C 主語と述語の関係にある D 動詞の後に目的語をおく E 前の漢字が後の漢字を修飾する

풀이: 読(읽다) + 書(책) → "書を読む(책을 읽다)" 뒷 글자(書)가 앞 글자 동사(読)의 목적어 → D. 동사+목적어

정답: D

가장 헷갈리는 구분: 유의어 vs 수식

이 두 패턴이 가장 혼동됩니다. 핵심 구분법:

유의어 vs 수식 판별법

유의어: 두 글자가 대등한 관계. "AもBも同じ(A도 B도 같다)"로 읽힘.
　例: 寒冷 → "寒も冷もつめたい(차갑다)" → 유의어

수식: 한쪽이 다른 쪽을 꾸며줌. "AいB(A한 B)"로 읽힘.
　例: 強風 → "強い風(강한 바람)" → 수식

⚠️ 함정 — 유의어인가 수식인가?

「救援」→ A(유의어) or E(수식)?

판별: 救(구하다) + 援(돕다) "救も援もたすける(둘 다 돕다)" → 대등한 관계 → A. 유의어 ✓ 만약 "救う援(구하는 도움)"이면 수식이지만, 두 글자 모두 동사이므로 유의어.

정답: A 유의어 결합
⚠️ 함정 — 주술인가 동사+목적어인가?

「地震」→ C(주술) or D(동사+목적어)?

판별: 地(땅이) + 震(흔들리다) "地が震える(땅이 흔들린다)" → 앞이 주어, 뒤가 서술어 → C. 주술 ✓ "地を震わす(땅을 흔들다)"로 읽으면 D가 되지만, 地震은 "땅이 흔들리는 현상"이므로 주술이 자연스러움.

정답: C 주술 관계
