SPI 비언어 — 유형 ⑤

집합·표 읽기 (集合·図表)

벤 다이어그램으로 집합 관계를 파악하고, 표·그래프에서 증가율·구성비를 계산하는 유형. 특히 표 읽기는 테스트센터에서 최빈출급입니다. 공식 자체는 단순하지만 "분모를 뭘로 잡느냐"에서 실수가 납니다.

이 유형의 핵심은 머릿속이 아닌 종이 위에서 푸는 것입니다. 집합은 벤 다이어그램, 표 읽기는 필요한 숫자에만 집중하는 것이 시간을 절약하는 방법입니다.

2그룹 집합 — 벤 다이어그램

"영어를 할 수 있는 사람 60명, 중국어를 할 수 있는 사람 40명, 양쪽 다 20명. 어느 쪽도 못하는 사람은?" 같은 유형입니다.

합집합 공식
A∪B = A + B - A∩B
A 또는 B = A + B - (양쪽 다). 중복을 한 번 빼야 함!
벤 다이어그램 예시: 100명, 영어 60, 중국어 40, 양쪽 20
영어 (60)
중국어 (40)
40
20
20
밖: 20명

영어만=40, 양쪽=20, 중국어만=20, 어느 쪽도 아님=20

⚠️ 함정 — 3그룹 집합이 나오면

A, B, C 3개 그룹이 나오면 포함배제 원리(包含排除)를 써야 합니다: |A∪B∪C| = |A|+|B|+|C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C| 3개 겹치는 부분부터 채워 나가는 것이 핵심.

실전에서는 3그룹까지 나오는 경우가 있지만 빈도는 낮습니다. 2그룹을 확실하게 풀 수 있으면 3그룹도 같은 원리의 확장입니다.

3그룹은 "가장 안쪽(3개 겹침)부터" 채우기!

표 읽기 — 증가율과 구성비

표나 그래프에서 숫자를 읽고 계산하는 유형. 테스트센터에서 특히 빈출입니다. 핵심 공식은 두 개:

증가율 (前年比)
증가율 = (이후 - 이전) / 이전 × 100%
주의: 분모는 반드시 "이전 값". 500→600이면 (600-500)/500 = 20%
구성비 (構成比)
구성비 = 해당 항목 / 합계 × 100%
전체 800만엔 중 개발부 200만엔 → 200/800 = 25%
⚠️ 증가율의 최대 함정

A상점: 500→600 (증가액 100) B상점: 800→920 (증가액 120) "증가액"이 큰 B가 증가율도 높다?

A: 100/500 = 20% / B: 120/800 = 15% → 증가"액"은 B가 크지만, 증가"율"은 A가 높다! 분모(기준)가 다르면 절대액과 비율은 역전될 수 있음.

증가"액"과 증가"율"을 절대 혼동하지 말 것!

자료 읽기 / 장문 읽기 계산 — 테스트센터 전용

테스트센터에서는 그래프·차트를 보고 읽는 "자료 읽기(資料の読み取り)"와, 긴 조건문을 읽고 계산하는 "장문 읽기 계산(長文読み取り計算)"이 추가로 출제됩니다.

접근법: 문제를 먼저 읽고 필요한 숫자만 표에서 찾는다. 표 전체를 읽으면 시간 낭비.

자주 하는 실수

실수 1: 합집합에서 중복을 안 빼고 단순 합산. A+B가 전체를 넘으면 반드시 중복이 있음!

실수 2: 증가율 계산에서 분모를 "이후 값"으로 잡음. 분모는 항상 "이전 값(기준)"!

실수 3: 표 문제에서 전체 표를 다 읽느라 시간 소모. 문제를 먼저 읽고 필요한 셀만 찾을 것.

Q01 — 2그룹 집합
영어·중국어 설문
100人にアンケート。英語ができる人は60人、中国語ができる人は40人、両方できる人は20人。英語も中国語もできない人は何人か。
100명 설문. 영어 60명, 중국어 40명, 양쪽 20명. 어느 쪽도 못하는 사람은?
풀이
합집합 = 60+40-20 = 80명 어느 쪽도 아님 = 100-80 = 20명
정답: 20명
Q02 — 집합 (영어만)
영어만 할 수 있는 사람
Q01と同じ条件で、英語だけができる人（中国語はできない人）は何人か。
Q01과 같은 조건. 영어만(중국어 불가) 할 수 있는 사람은?
풀이
영어만 = 영어 전체 - 양쪽 = 60 - 20 = 40명
정답: 40명
Q03 — 증가율
매출 증가율 비교
A店の売上：2023年500万円、2024年600万円。B店の売上：2023年800万円、2024年920万円。売上の増加率が高いのはどちらか。
A상점 500→600만엔. B상점 800→920만엔. 증가율이 높은 쪽은?
풀이
A: (600-500)/500 = 20% B: (920-800)/800 = 15% A > B
정답: A상점 (20%)
Q04 — 구성비
부문별 매출 비율
営業部240万円、企画部160万円、開発部200万円、総務部80万円、人事部120万円。開発部の売上が全体に占める割合は何%か。
영업 240, 기획 160, 개발 200, 총무 80, 인사 120(만엔). 개발부 비율은?
풀이
합계 = 240+160+200+80+120 = 800만엔 개발 비율 = 200/800 × 100 = 25%
정답: 25%
Q05 — 집합 (적어도)
적어도 1과목 합격
50人が数学と英語の試験を受けた。数学合格者32人、英語合格者28人、両方不合格者5人。両方合格した人は何人か。
50명이 수학·영어 시험. 수학 합격 32명, 영어 합격 28명, 둘 다 불합격 5명. 둘 다 합격한 사람은?
풀이
적어도 1과목 합격 = 50-5 = 45명 수학+영어-양쪽 = 45 32+28-양쪽 = 45 양쪽 = 60-45 = 15명
정답: 15명
Q06 — 표 읽기 (감소율)
전년 대비 감소
ある商品の売上が2023年は400万円、2024年は340万円だった。売上の減少率は何%か。
상품 매출 2023년 400만엔 → 2024년 340만엔. 감소율은?
풀이
감소율 = (400-340)/400 × 100 = 60/400 × 100 = 15%
정답: 15%
Q07 — 표 읽기 (복합)
2년간 총 증가액
A部門：2022年300万、2023年350万、2024年400万。B部門：2022年200万、2023年180万、2024年250万。2022年から2024年にかけて、売上が増加した部門の合計増加額はいくらか。
A부문 300→350→400. B부문 200→180→250. 2022~2024 매출 증가 부문의 합계 증가액은?
풀이
A: 400-300 = +100만 (증가 ✓) B: 250-200 = +50만 (증가 ✓) 합계 증가액 = 100+50 = 150만엔
정답: 150만엔
Q08 — 집합 응용
3종류 음료
60人にアンケート。コーヒーを飲む人35人、紅茶を飲む人25人、両方飲む人15人。コーヒーも紅茶も飲まない人は何人か。ただし、3種類目の飲み物は考えない。
60명 설문. 커피 35명, 홍차 25명, 양쪽 15명. 둘 다 안 마시는 사람은?
풀이
합집합 = 35+25-15 = 45명 둘 다 아님 = 60-45 = 15명
정답: 15명
Q09 — 득점 범위
최고점의 최솟값
5人の平均点が60点で、最低点が40点だった。最高点として考えられる最小の値はいくらか。ただし全員の点数は整数とする。
5명 평균 60점, 최저 40점. 최고점의 최솟값은? (모두 정수)
풀이
합계 = 60×5 = 300점 최고점 최소화 → 나머지를 최대화 나머지 3명을 최고점과 같은 값(X)으로 놓으면: 40 + X + X + X + X = 300 → 4X = 260 → X = 65 최고점 ≥ 65이므로 최솟값 = 65
정답: 65점
Q10 — 표 읽기 (비율 비교)
점유율 변화
A社の市場シェア：2023年は全体1,000億円中300億円、2024年は全体1,200億円中330億円。A社のシェアは上がったか下がったか。
A사 시장 점유율: 2023년 전체 1000억 중 300억, 2024년 전체 1200억 중 330억. 점유율은 올랐나 내렸나?
풀이
2023: 300/1000 = 30% 2024: 330/1200 = 27.5% 매출 자체는 300→330으로 올랐지만, 점유율은 30→27.5로 내렸다. ※ 시장 전체가 더 빠르게 성장한 경우.
정답: 점유율은 내렸다 (30% → 27.5%)
