SPI 비언어 — 유형 ⑤

집합·표 읽기 (集合·図表)

벤 다이어그램으로 집합 관계를 파악하고, 표·그래프에서 증가율·구성비를 계산하는 유형. 특히 표 읽기는 테스트센터에서 최빈출급입니다. 공식 자체는 단순하지만 "분모를 뭘로 잡느냐"에서 실수가 납니다.

테센 ★ 빈출웹테 ○ 출제
강의
집합과 표 읽기 — 시각화가 전부
集合·図表 — 可視化がすべて

이 유형의 핵심은 머릿속이 아닌 종이 위에서 푸는 것입니다. 집합은 벤 다이어그램, 표 읽기는 필요한 숫자에만 집중하는 것이 시간을 절약하는 방법입니다.

2그룹 집합 — 벤 다이어그램

"영어를 할 수 있는 사람 60명, 중국어를 할 수 있는 사람 40명, 양쪽 다 20명. 어느 쪽도 못하는 사람은?" 같은 유형입니다.

합집합 공식
A∪B = A + B - A∩B
A 또는 B = A + B - (양쪽 다). 중복을 한 번 빼야 함!
벤 다이어그램 예시: 100명, 영어 60, 중국어 40, 양쪽 20
영어 (60)
중국어 (40)
40
20
20
밖: 20명

영어만=40, 양쪽=20, 중국어만=20, 어느 쪽도 아님=20

⚠️ 함정 — 3그룹 집합이 나오면

A, B, C 3개 그룹이 나오면 포함배제 원리(包含排除)를 써야 합니다: |A∪B∪C| = |A|+|B|+|C| - |A∩B| - |A∩C| - |B∩C| + |A∩B∩C| 3개 겹치는 부분부터 채워 나가는 것이 핵심.

실전에서는 3그룹까지 나오는 경우가 있지만 빈도는 낮습니다. 2그룹을 확실하게 풀 수 있으면 3그룹도 같은 원리의 확장입니다.

3그룹은 "가장 안쪽(3개 겹침)부터" 채우기!

표 읽기 — 증가율과 구성비

표나 그래프에서 숫자를 읽고 계산하는 유형. 테스트센터에서 특히 빈출입니다. 핵심 공식은 두 개:

증가율 (前年比)
증가율 = (이후 - 이전) / 이전 × 100%
주의: 분모는 반드시 "이전 값". 500→600이면 (600-500)/500 = 20%
구성비 (構成比)
구성비 = 해당 항목 / 합계 × 100%
전체 800만엔 중 개발부 200만엔 → 200/800 = 25%
⚠️ 증가율의 최대 함정

A상점: 500→600 (증가액 100) B상점: 800→920 (증가액 120) "증가액"이 큰 B가 증가율도 높다?

A: 100/500 = 20% / B: 120/800 = 15% → 증가"액"은 B가 크지만, 증가"율"은 A가 높다! 분모(기준)가 다르면 절대액과 비율은 역전될 수 있음.

증가"액"과 증가"율"을 절대 혼동하지 말 것!

자료 읽기 / 장문 읽기 계산 — 테스트센터 전용

테스트센터에서는 그래프·차트를 보고 읽는 "자료 읽기(資料の読み取り)"와, 긴 조건문을 읽고 계산하는 "장문 읽기 계산(長文読み取り計算)"이 추가로 출제됩니다.

접근법: 문제를 먼저 읽고 필요한 숫자만 표에서 찾는다. 표 전체를 읽으면 시간 낭비.

자주 하는 실수

실수 1: 합집합에서 중복을 안 빼고 단순 합산. A+B가 전체를 넘으면 반드시 중복이 있음!

실수 2: 증가율 계산에서 분모를 "이후 값"으로 잡음. 분모는 항상 "이전 값(기준)"!

실수 3: 표 문제에서 전체 표를 다 읽느라 시간 소모. 문제를 먼저 읽고 필요한 셀만 찾을 것.

Q01 — 2그룹 집합
영어·중국어 설문
100人にアンケート。英語ができる人は60人、中国語ができる人は40人、両方できる人は20人。英語も中国語もできない人は何人か。
100명 설문. 영어 60명, 중국어 40명, 양쪽 20명. 어느 쪽도 못하는 사람은?
풀이
합집합 = 60+40-20 = 80명 어느 쪽도 아님 = 100-80 = 20명
정답: 20명
Q02 — 집합 (영어만)
영어만 할 수 있는 사람
Q01と同じ条件で、英語だけができる人（中国語はできない人）は何人か。
Q01과 같은 조건. 영어만(중국어 불가) 할 수 있는 사람은?
풀이
영어만 = 영어 전체 - 양쪽 = 60 - 20 = 40명
정답: 40명
Q03 — 증가율
매출 증가율 비교
A店の売上：2023年500万円、2024年600万円。B店の売上：2023年800万円、2024年920万円。売上の増加率が高いのはどちらか。
A상점 500→600만엔. B상점 800→920만엔. 증가율이 높은 쪽은?
풀이
A: (600-500)/500 = 20% B: (920-800)/800 = 15% A > B
정답: A상점 (20%)
Q04 — 구성비
부문별 매출 비율
営業部240万円、企画部160万円、開発部200万円、総務部80万円、人事部120万円。開発部の売上が全体に占める割合は何%か。
영업 240, 기획 160, 개발 200, 총무 80, 인사 120(만엔). 개발부 비율은?
풀이
합계 = 240+160+200+80+120 = 800만엔 개발 비율 = 200/800 × 100 = 25%
정답: 25%
Q05 — 집합 (적어도)
적어도 1과목 합격
50人が数学と英語の試験を受けた。数学合格者32人、英語合格者28人、両方不合格者5人。両方合格した人は何人か。
50명이 수학·영어 시험. 수학 합격 32명, 영어 합격 28명, 둘 다 불합격 5명. 둘 다 합격한 사람은?
풀이
적어도 1과목 합격 = 50-5 = 45명 수학+영어-양쪽 = 45 32+28-양쪽 = 45 양쪽 = 60-45 = 15명
정답: 15명
Q06 — 표 읽기 (감소율)
전년 대비 감소
ある商品の売上が2023年は400万円、2024年は340万円だった。売上の減少率は何%か。
상품 매출 2023년 400만엔 → 2024년 340만엔. 감소율은?
풀이
감소율 = (400-340)/400 × 100 = 60/400 × 100 = 15%
정답: 15%
Q07 — 표 읽기 (복합)
2년간 총 증가액
A部門：2022年300万、2023年350万、2024年400万。B部門：2022年200万、2023年180万、2024年250万。2022年から2024年にかけて、売上が増加した部門の合計増加額はいくらか。
A부문 300→350→400. B부문 200→180→250. 2022~2024 매출 증가 부문의 합계 증가액은?
풀이
A: 400-300 = +100만 (증가 ✓) B: 250-200 = +50만 (증가 ✓) 합계 증가액 = 100+50 = 150만엔
정답: 150만엔
Q08 — 집합 응용
3종류 음료
60人にアンケート。コーヒーを飲む人35人、紅茶を飲む人25人、両方飲む人15人。コーヒーも紅茶も飲まない人は何人か。ただし、3種類目の飲み物は考えない。
60명 설문. 커피 35명, 홍차 25명, 양쪽 15명. 둘 다 안 마시는 사람은?
풀이
합집합 = 35+25-15 = 45명 둘 다 아님 = 60-45 = 15명
정답: 15명
Q09 — 득점 범위
최고점의 최솟값
5人の平均点が60点で、最低点が40点だった。最高点として考えられる最小の値はいくらか。ただし全員の点数は整数とする。
5명 평균 60점, 최저 40점. 최고점의 최솟값은? (모두 정수)
풀이
합계 = 60×5 = 300점 최고점 최소화 → 나머지를 최대화 나머지 3명을 최고점과 같은 값(X)으로 놓으면: 40 + X + X + X + X = 300 → 4X = 260 → X = 65 최고점 ≥ 65이므로 최솟값 = 65
정답: 65점
Q10 — 표 읽기 (비율 비교)
점유율 변화
A社の市場シェア：2023年は全体1,000億円中300億円、2024年は全体1,200億円中330億円。A社のシェアは上がったか下がったか。
A사 시장 점유율: 2023년 전체 1000억 중 300억, 2024년 전체 1200억 중 330억. 점유율은 올랐나 내렸나?
풀이
2023: 300/1000 = 30% 2024: 330/1200 = 27.5% 매출 자체는 300→330으로 올랐지만, 점유율은 30→27.5로 내렸다. ※ 시장 전체가 더 빠르게 성장한 경우.
정답: 점유율은 내렸다 (30% → 27.5%)

P01 — 2집합 기본 기초
개·고양이 설문
30人にアンケートを取った。犬が好き18人、猫が好き14人、両方好き6人。どちらも好きでない人は何人か。
30명에게 설문. 개를 좋아하는 사람 18명, 고양이 14명, 둘 다 6명. 둘 다 좋아하지 않는 사람은?
포함-배제
적어도 하나 = 18 + 14 - 6 = 26명 둘 다 안 좋아함 = 30 - 26 = 4명
정답: 4명
P02 — 집합 (한쪽만) 기초
개만 좋아하는 사람
P01と同じ条件で、犬だけが好きな人（猫は好きでない人）は何人か。
P01과 같은 조건. 개만 좋아하는 사람(고양이는 ×)은?
풀이
개만 = 개 전체 - 둘 다 = 18 - 6 = 12명
정답: 12명
P03 — 증가율 기본 기초
매출 증가율
ある店の昨年の売上は200万円、今年は240万円。売上の増加率は何%か。
어떤 가게의 작년 매출 200만엔, 올해 240만엔. 증가율은?
풀이
증가율 = (240-200)/200 × 100 = 40/200 × 100 = 20%
정답: 20%
P04 — 구성비 기본 기초
비율 계산
全体の売上が500万円で、商品Aの売上が125万円。商品Aの構成比は何%か。
전체 매출 500만엔, 상품A 매출 125만엔. 상품A의 구성비는?
풀이
125/500 × 100 = 25%
정답: 25%
P05 — 대우 명제 기초
논리 기본
「雨が降れば試合は中止になる」の対偶はどれか。 (A) 試合が中止にならなければ雨は降らなかった (B) 雨が降らなければ試合は中止にならない (C) 試合が中止になれば雨が降った
"비가 오면 시합은 중지된다"의 대우는? (A) 시합이 중지되지 않으면 비는 오지 않았다 (B) 비가 오지 않으면 시합은 중지되지 않는다 (C) 시합이 중지되면 비가 왔다
풀이
원래: P→Q (비→중지) 대우: ¬Q→¬P (중지 아님→비 아님) = (A) (B)는 역의 부정, (C)는 역
정답: (A)
P06 — 표 읽기 (증감) 실전
3년간 매출 비교
ある会社の売上：2022年 800万円、2023年 920万円、2024年 874万円。 2022→2023年の増加率と、2023→2024年の減少率をそれぞれ求めよ。
어떤 회사 매출: 2022년 800만, 2023년 920만, 2024년 874만. 2022→2023 증가율과 2023→2024 감소율을 각각 구하라.
풀이
22→23 증가율: (920-800)/800 × 100 = 15% 23→24 감소율: (920-874)/920 × 100 = 46/920 × 100 = 5%
정답: 증가율 15%, 감소율 5%
P07 — 집합 (여사건) 실전
적어도 1과목 합격
50人が数学と英語の試験を受けた。数学合格35人、英語合格28人、両方不合格5人。両方合格した人は何人か。
50명이 수학·영어 시험을 봤다. 수학 합격 35명, 영어 합격 28명, 둘 다 불합격 5명. 둘 다 합격한 사람은?
풀이
적어도 하나 합격 = 50 - 5 = 45명 35 + 28 - x = 45 → x = 18
정답: 18명
P08 — 표 읽기 (점유율) 실전
시장 점유율
ある市場の全体は1,000億円。A社250億円、B社200億円、C社150億円、その他400億円。 A社の市場シェアが前年の28%から変化した。シェアは上がったか下がったか。
시장 전체 1,000억엔. A사 250억, B사 200억, C사 150억, 기타 400억. A사 점유율이 전년 28%에서 변동. 올랐나 내렸나?
풀이
올해 A사 점유율 = 250/1000 × 100 = 25% 전년 28% → 25%로 하락
정답: 내렸다 (28%→25%, 3%p 하락)
P09 — 논리 추론 실전
필요·충분 조건
「xが4の倍数ならばxは偶数である」 この命題の逆·裏·対偶のうち、必ず真であるものはどれか。
"x가 4의 배수이면 x는 짝수이다" 이 명제의 역·이·대우 중 반드시 참인 것은?
풀이
원래(참): 4배수→짝수 대우(항상 원래와 진리값 동일): 짝수 아님→4배수 아님 = 홀수→4배수 아님 → 참 ✓ 역: 짝수→4배수? x=6은 짝수지만 4배수× → 거짓 이: 4배수 아님→짝수 아님? x=6 → 거짓
정답: 대우 (「偶数でなければ4の倍数でない」)
P10 — 득점 범위 실전
평균과 범위
5人のテストの平均点が72点。最高点が95点、最低点が40点。残り3人の平均点は何点か。
5명 시험 평균 72점. 최고 95점, 최저 40점. 나머지 3명의 평균은?
풀이
총점 = 72 × 5 = 360점 나머지 3명 합 = 360 - 95 - 40 = 225점 평균 = 225/3 = 75점
정답: 75점
P11 — 표 읽기 (복합) 실전
2부문 비교
A部門：2023年 売上150万円（利益率20%）、B部門：2023年 売上200万円（利益率15%）。 利益額が大きいのはどちらか。差額はいくらか。
A부문: 매출 150만(이익률 20%). B부문: 매출 200만(이익률 15%). 이익액이 큰 쪽은? 차액은?
풀이
A 이익 = 150 × 0.20 = 30만엔 B 이익 = 200 × 0.15 = 30만엔 → 동일! 차액 = 0
정답: 같다 (각 30만엔, 차액 0)
P12 — 3집합 고난도
3과목 설문
60人に英語·数学·国語の好き嫌いを聞いた。英語30人、数学25人、国語20人が好き。英数両方10人、英国両方8人、数国両方6人、3科目全部好き3人。どれも好きでない人は何人か。
60명에게 영어·수학·국어 선호 조사. 영어 30, 수학 25, 국어 20명 좋아함. 영+수 10, 영+국 8, 수+국 6, 3과목 전부 3명. 어느 것도 좋아하지 않는 사람은?
3집합 포함-배제
|A∪B∪C| = |A|+|B|+|C|-|A∩B|-|A∩C|-|B∩C|+|A∩B∩C| = 30+25+20-10-8-6+3 = 54명
결론
어느 것도 × = 60 - 54 = 6명
정답: 6명
P13 — 표 읽기 (CAGR) 고난도
2년 연평균 성장률
売上が2022年に100万円、2024年に121万円になった。年平均成長率（CAGR）は何%か。
매출이 2022년 100만엔에서 2024년 121만엔. 연평균 성장률(CAGR)은?
풀이
2년간: 100 × (1+r)² = 121 (1+r)² = 1.21 → 1+r = 1.1 → r = 0.1
정답: 10%
P14 — 집합+논리 고난도
확실히 말할 수 있는 것
100人のうち、A資格保有者60人、B資格保有者50人。次のうち確実に言えるのはどれか。 (ア) 両方持つ人は少なくとも10人いる (イ) 両方持つ人は最大50人である (ウ) どちらも持たない人は最大40人である
100명 중 A자격 보유 60명, B자격 보유 50명. 확실히 말할 수 있는 것은? (ア) 양쪽 보유자는 적어도 10명 (イ) 양쪽 보유자는 최대 50명 (ウ) 양쪽 미보유는 최대 40명
풀이
(ア) 최소 양쪽 = A+B-전체 = 60+50-100 = 10 → 적어도 10명 ✓ (イ) 최대 양쪽 = min(60,50) = 50 → 최대 50명 ✓ (ウ) 양쪽 미보유 최대: 양쪽 보유 최소(=10)일 때 → 적어도 하나 보유 = 60+50-10=100 → 미보유 0명. 미보유 최대는 양쪽 보유 최대(=50)일 때: 적어도 하나 = 60+50-50=60 → 미보유=40 ✓
정답: 3개 모두 확실히 말할 수 있다
P15 — 표 읽기+추론 고난도
누락 데이터 추정
3部門の売上（単位：万円） 　　2023年　2024年 A部門　120　　? B部門　?　　　160 C部門　80　　　90 全体合計：2023年350万円、2024年400万円。A部門2024年とB部門2023年の売上を求めよ。
3부문 매출(만엔): A 2023=120/2024=?, B 2023=?/2024=160, C 2023=80/2024=90. 전체 합계: 2023년 350만, 2024년 400만. A 2024년과 B 2023년은?
풀이
2023년: A(120)+B+C(80)=350 → B=150만엔 2024년: A+B(160)+C(90)=400 → A=150만엔
정답: A 2024년=150만엔, B 2023년=150만엔
