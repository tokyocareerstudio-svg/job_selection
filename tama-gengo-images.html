취지판정 (趣旨判定)

400-600자의 장문을 읽고, 설문문이 A(필자가 가장 말하고 싶은 것=취지) B(본문에 쓰여 있지만 취지는 아님) C(본문과 관계없음) 중 어디에 해당하는지 판별하는 유형. GAB형보다 시간이 짧고(10분/32문), 「취지인가 아닌가」를 구분해야 하므로 A와 B의 경계가 가장 어렵습니다.

32문 (8장문 x 4문) / 10분
강의
IMAGES형 취지판정이란?
IMAGES形式の趣旨判定 — 筆者が一番言いたいことを見抜く

IMAGES 페이퍼 시험에서 유래한 형식으로, 玉手箱 언어에서 GAB형 다음으로 자주 출제됩니다. GAB형(15분)보다 시간이 5분 짧은 10분에 같은 32문을 풀어야 하므로, 1장문당 약 1분 15초밖에 없습니다. 시간 압박이 훨씬 심합니다.

3택 선택지 (전 문제 공통)

A. 필자가 가장 호소하고 싶은 것(취지)이 서술되어 있다
→ 본문 전체의 결론, 주장, 메인 메시지에 해당

B. 본문에 쓰여 있지만, 가장 호소하고 싶은 것은 아니다
→ 본문에 언급은 있으나 부수적 정보, 배경, 예시에 불과

C. 본문과 관계없는 것이 쓰여 있다
→ 본문에 전혀 언급되지 않은 내용

핵심 규칙

화면 상단에 「설문문에는 A와 C에 해당하는 것이 반드시 1개 이상 포함되어 있다」는 주의문이 표시됩니다. 즉, 4문 중 A가 최소 1개, C가 최소 1개는 반드시 있습니다. 전부 B로 하거나, A가 0개인 응답은 불가능합니다.

GAB형과의 결정적 차이

GAB형은 「맞다/틀리다/판단불가」라는 논리적 판별이지만, IMAGES형은 「취지인가 아닌가」라는 중요도 판별입니다. 본문에 쓰여 있어도 메인 주장이 아니면 B입니다. 이 A와 B의 구분이 IMAGES형의 최대 난관입니다.

📝 미니 예제

【본문 (발췌)】 「就職活動において、自己分析は欠かせないプロセスである。自分の強みや価値観を明確にすることで、企業選びの軸が定まり、面接でも一貫した回答ができるようになる。しかし、自己分析に時間をかけすぎて行動が遅れるケースも少なくない。大切なのは、完璧な自己分析を目指すことではなく、ある程度の方向性が見えた段階で企業研究や選考に進むことである。」 【설문문1】自己分析は就職活動において重要である。 【설문문2】自己分析よりも行動を優先すべきである。 【설문문3】面接対策にはグループディスカッションの練習が有効である。 【설문문4】完璧な自己分析を目指すのではなく、適度な段階で行動に移るべきだ。

풀이: 설문1: 본문 서두에 「自己分析は欠かせない」라고 쓰여 있음 → 본문에 있지만, 필자의 최종 주장은 아님 = B 설문2: 본문에 자기분석보다 행동이 우선이라고는 하지 않았음. 자기분석은 필요하되 과하게 하지 말라는 것 = C 설문3: 그룹 디스커션 연습에 대한 언급 전혀 없음 = C 설문4: 본문 마지막 문장의 결론과 정확히 일치 = A (취지)

정답: 설문1=B / 설문2=C / 설문3=C / 설문4=A

A와 B를 구분하는 핵심 기법

A(취지)를 찾는 3가지 시그널

1. 결론 위치: 일본어 문장은 마지막에 결론을 놓는 경향. 「大切なのは」「重要なことは」「つまり」「結局」 뒤에 나오는 내용이 취지일 확률이 높음

2. 역접 뒤: 「しかし」「だが」「一方で」 뒤에 나오는 주장이 필자의 진짜 의견인 경우가 많음

3. 강조 표현: 「べきだ」「必要がある」「不可欠だ」 같은 당위 표현이 포함된 문장

B(본문에 있지만 취지 아님)가 되는 전형

• 배경 설명 — 「近年」「従来」로 시작하는 상황 서술
• 구체적 예시 — 「例えば」 뒤의 설명
• 부수적 사실 — 취지를 뒷받침하는 데이터나 통계
• 일반론 — 「と言われている」 같은 일반적 견해

시간 배분 가이드

32문 / 10분 = 1문당 약 19초

• 장문 1개 + 설문 4개 = 약 1분 15초 이내
• 전략: 본문 마지막 2-3문장만 먼저 읽고 취지(A) 후보를 파악 → 설문과 대조
• C를 먼저 제거 (본문에 없는 내용은 즉시 C) → 나머지에서 A/B 판별
• A와 B에서 30초 이상 고민하면 직감으로 선택하고 넘기기 (오류율 미측정)

연습 문제
IMAGES형 취지판정 예제 10문제
IMAGES形式の趣旨判定 練習問題10問

본문을 읽고 각 설문문에 A(취지) B(본문에 있지만 취지 아님) C(관계없음) 중 하나를 선택하세요.

본문 1 (Q01-Q04)
팀워크와 리더십
企業が求める人材像として「チームワーク」が挙げられることは多い。確かにチームで協力して成果を出す能力は重要である。しかし、単に周囲に合わせるだけの協調性ではなく、自分の意見を持ちながらもチーム全体の目標に向けて調整できる力こそが、真のチームワークである。特に新入社員に求められるのは、指示を待つだけでなく、自ら考えて提案する主体性を持ったうえでの協調性だ。
Q01
설문
企業が求める人材像としてチームワークが重視されている。
풀이
본문 서두에 「チームワークが挙げられることは多い」라고 쓰여 있음. 사실이지만 도입부의 배경 설명이지 필자의 최종 주장은 아님.
정답: B — 본문에 있지만 취지 아님
Q02
설문
真のチームワークとは、自分の意見を持ちながらチーム目標に向けて調整できる力である。
풀이
「しかし」 뒤에 나오는 핵심 주장. 「〜こそが、真のチームワークである」라는 강조 구문으로 필자의 메인 메시지.
정답: A — 취지
Q03
설문
新入社員にはリーダーシップ研修が必要である。
풀이
본문에 리더십 연수에 대한 언급은 전혀 없음.
정답: C — 관계없음
Q04
설문
新入社員には、指示を待つだけでなく主体性を持った協調性が求められる。
풀이
본문 마지막 문장의 내용과 일치. 취지를 구체적으로 신입사원에게 적용한 부연 설명. A와 헷갈릴 수 있지만, 이 문장은 취지(Q02)의 구체적 적용례이므로 B.
정답: B — 본문에 있지만 취지 아님 (취지의 구체적 적용례)
본문 2 (Q05-Q08)
워크라이프밸런스
長時間労働の是正は、日本企業にとって長年の課題である。近年、働き方改革関連法の施行により、残業時間の上限規制が設けられた。また、在宅勤務やフレックスタイム制度を導入する企業も増加している。しかし、制度を整えるだけでは不十分であり、重要なのは従業員一人ひとりが効率的な働き方を自ら考え、実践することである。制度に頼るだけでなく、個人の意識改革が働き方改革の本質だと言えるだろう。
Q05
설문
日本では働き方改革関連法により、残業時間の上限規制が設けられた。
풀이
본문에 직접 쓰여 있는 사실이지만, 배경 설명에 해당하며 필자의 주장은 아님.
정답: B — 본문에 있지만 취지 아님
Q06
설문
働き方改革の本質は、制度整備だけでなく個人の意識改革にある。
풀이
본문 마지막 문장 「個人の意識改革が働き方改革の本質だ」와 정확히 일치. 필자의 결론.
정답: A — 취지
Q07
설문
在宅勤務の導入率は、大企業より中小企業の方が高い。
풀이
본문에는 재택근무 도입 기업이 증가하고 있다는 것만 있고, 대기업/중소기업 간 비교는 전혀 없음.
정답: C — 관계없음
Q08
설문
フレックスタイム制度を導入する企業が増えている。
풀이
본문에 「フレックスタイム制度を導入する企業も増加している」라고 직접 쓰여 있음. 그러나 배경 정보이지 취지는 아님.
정답: B — 본문에 있지만 취지 아님
본문 3 (Q09-Q10)
데이터 활용과 의사결정
ビジネスにおけるデータ活用の重要性が叫ばれて久しい。多くの企業がBI（ビジネスインテリジェンス）ツールを導入し、データに基づいた意思決定を推進している。だが、データはあくまで判断材料の一つであり、最終的な意思決定には経験や直感も不可欠である。データと人間の判断を適切に組み合わせることが、これからの経営に求められる姿勢だろう。
Q09
설문
データに基づいた意思決定だけで十分であり、経験や直感は不要だ。
풀이
본문에 「経験や直感も不可欠」이라고 되어 있으므로 「불요」라는 설문은 관계없는 것이 아니라 본문과 반대. 그러나 IMAGES형에서 C는 「관계없음」이지 「틀림」이 아님. 본문과 내용이 연결되어 있으므로 B가 아니라... 실은 IMAGES형에는 B도 아님(취지가 아닌 것은 B이지만 이건 본문과 반대). IMAGES형에서 본문의 내용과 반대인 것은 「본문에 관계없는 것」으로 분류 = C.
정답: C — 관계없음 (본문의 주장과 반대되는 내용은 C)
Q10
설문
データと人間の判断を組み合わせることが、これからの経営に求められる。
풀이
본문 마지막 문장 「データと人間の判断を適切に組み合わせることが、これからの経営に求められる姿勢」와 정확히 일치. 필자의 결론.
정답: A — 취지
