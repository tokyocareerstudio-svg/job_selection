두 단어의 관계 (二語の関係)

두 단어의 관계를 파악하고, 같은 관계를 가진 쌍을 찾는 유형. 8가지 관계 유형을 암기하고 "두 단어로 문장 만들기" 기법을 익히면 빠르게 풀 수 있습니다.

테센 ★ 전용
강의
두 단어의 관계란?
二語の関係 — 二つの言葉で文を作れば関係が見える

이 유형은 제시된 두 단어의 관계(동의어, 반의어, 포함 관계 등)를 파악한 뒤, 선택지에서 같은 관계를 가진 쌍을 고르는 문제입니다. 테스트센터 전용이며, 1문제 10~20초로 빠르게 풀어야 합니다.

핵심 풀이법은 단 하나: "두 단어로 짧은 문장을 만들어보기". 제시어로 문장을 만들고, 선택지로도 같은 구조의 문장을 만들어서 일치하는 것을 찾습니다.

8대 관계 유형 (반드시 암기)

① 동의어: A와 B는 같은 뜻 (逆境 = 苦境)
② 반의어: A와 B는 반대 뜻 (慎重 ↔ 軽率)
③ 포함 (상위↔하위): A는 B의 일종 (スポーツ ⊃ 野球)
④ 역할: A는 B에서 일한다 (医者 → 病院)
⑤ 원료·재료: A로 B를 만든다 (米 → 酒)
⑥ 세트: A와 B는 함께 사용 (針 + 糸)
⑦ 원인과 결과: A하면 B가 된다 (勉強 → 合格)
⑧ 목적: A는 B를 위해 존재 (冷蔵庫 → 保存)

풀이 기법 — "문장 만들기" 상세 해설

이 풀이법은 3단계로 진행합니다.

문장 만들기 3단계

1단계: 제시된 두 단어로 자연스러운 한 문장을 만든다
　예: "医者 : 病院" → "의사는 병원에서 일한다"

2단계: 각 선택지로도 같은 방식으로 문장을 만들어본다
　ア "교사는 학교에서 일한다" / イ "책은 도서관에 있다" / ...

3단계: 제시어 문장과 같은 구조(~는 ~에서 일한다)인 선택지를 고른다

📝 미니 예제 — 역할 관계

医者 : 病院 ア　教師 : 学校　　イ　本 : 図書館 ウ　犬 : 動物　　エ　赤 : 色

풀이: "의사는 병원에서 일한다" → 역할(장소) 관계 ア "교사는 학교에서 일한다" → 같은 역할 관계 ✓ イ "책은 도서관에 있다" → 소속이지 역할이 아님 ウ "개는 동물의 일종" → 포함 / エ "빨강은 색의 일종" → 포함

정답: ア 教師 : 学校

헷갈리기 쉬운 유형 구분법

⚠️ 함정 — 포함 vs 역할 혼동

著者 : 本 ア　花 : 桜　　イ　画家 : 絵 ウ　机 : 椅子　　エ　春 : 秋

함정: "저자와 책"을 "책은 저자의 일종"(포함)으로 오해하기 쉽지만, "저자는 책을 쓴다" → 역할(행위) 관계. イ "화가는 그림을 그린다" → 같은 역할 관계 ✓ ア 벚꽃은 꽃의 일종 → 포함 / ウ 함께 사용 → 세트 / エ 반의어

정답: イ 画家 : 絵
⚠️ 함정 — 원인결과 vs 목적 혼동

練習 : 上達 ア　冷蔵庫 : 保存　　イ　雨 : 洪水 ウ　勉強 : 合格　　エ　鍵 : 錠前

함정: "연습하면 실력이 오른다" → 원인→결과. ア "냉장고는 보존을 위해 있다" → 목적 관계(인과 아님!) イ "비가 오면 홍수가 난다" → 원인→결과이지만 부정적 ウ "공부하면 합격한다" → 같은 원인→결과 구조 ✓

정답: ウ 勉強 : 合格
시간 관리

이 유형은 1문제 10~20초가 목표입니다. 문장 만들기에 익숙해지면 5초 안에도 풀 수 있습니다. 연습 포인트: 아래 유형별 빈출 예시에서 각 쌍으로 "문장 만들기"를 반복 연습하세요. 패턴이 보이기 시작하면 속도가 비약적으로 빨라집니다.

시간 관리 팁

1문제 10~20초 목표. 두 단어로 문장을 만들어 관계 파악 → 선택지도 같은 방식으로 문장 만들기. 연습할수록 패턴이 보여 5초만에도 풀 수 있습니다.

연습 문제
두 단어의 관계 예제 10문제
二語の関係の練習問題10問 — テストセンター専用

제시된 두 단어의 관계와 같은 관계의 쌍을 찾으세요. "문장 만들기"로 풀어보세요!

Q01
동의어
示された二語の関係と同じ関係のものを選びなさい。 逆境 : 苦境 ア　出版 : 刊行　　イ　上昇 : 下降 ウ　医者 : 病院　　エ　犬 : 動物
역경 : 고경 과 같은 관계는? ア 출판:간행 イ 상승:하강 ウ 의사:병원 エ 개:동물
풀이
逆境=苦境 → 동의어 관계. ア 出版=刊行 → 동의어 ✓ イ 상승↔하강 → 반의어 ウ 의사는 병원에서 일한다 → 역할 エ 개는 동물의 일종 → 포함
정답: ア 出版 : 刊行 (동의어)
Q02
반의어
慎重 : 軽率 ア　温暖 : 寒冷　　イ　教師 : 学校 ウ　原因 : 結果　　エ　鉛筆 : 消しゴム
신중 : 경솔 과 같은 관계는? ア 온난:한랭 イ 교사:학교 ウ 원인:결과 エ 연필:지우개
풀이
慎重↔軽率 → 반의어. ア 温暖↔寒冷 → 반의어 ✓ ウ 원인과 결과 → 인과관계(반의어 아님) エ 함께 쓰는 것 → 세트 관계
정답: ア 温暖 : 寒冷 (반의어)
Q03
포함 관계
スポーツ : 野球 ア　花 : 桜　　イ　机 : 椅子 ウ　著者 : 本　　エ　春 : 秋
스포츠 : 야구 와 같은 관계는? ア 꽃:벚꽃 イ 책상:의자 ウ 저자:책 エ 봄:가을
풀이
야구는 스포츠의 일종 → 포함 관계(상위:하위). ア 벚꽃은 꽃의 일종 → 포함 관계 ✓ イ 함께 사용 → 세트 ウ 저자가 책을 쓴다 → 역할 エ 봄↔가을 → 반의어
정답: ア 花 : 桜 (포함)
Q04
역할 관계
医者 : 病院 ア　教師 : 学校　　イ　赤 : 色 ウ　安価 : 廉価　　エ　上 : 下
의사 : 병원 과 같은 관계는? ア 교사:학교 イ 빨강:색 ウ 저렴:염가 エ 위:아래
풀이
의사는 병원에서 일한다 → 역할(장소). ア 교사는 학교에서 일한다 → 같은 역할 관계 ✓
정답: ア 教師 : 学校 (역할)
Q05
원료 관계
米 : 酒 ア　木 : 紙　　イ　画家 : 絵 ウ　朝 : 夜　　エ　果物 : りんご
쌀 : 술 과 같은 관계는? ア 나무:종이 イ 화가:그림 ウ 아침:밤 エ 과일:사과
풀이
쌀로 술을 만든다 → 원료 관계. ア 나무로 종이를 만든다 → 원료 관계 ✓ イ 화가가 그림을 그린다 → 역할 ウ 아침↔밤 → 반의어 エ 사과는 과일의 일종 → 포함
정답: ア 木 : 紙 (원료)
Q06
세트 관계
針 : 糸 ア　鍵 : 錠前　　イ　魚 : 海 ウ　大 : 小　　エ　作家 : 小説
바늘 : 실 과 같은 관계는? ア 열쇠:자물쇠 イ 물고기:바다 ウ 대:소 エ 작가:소설
풀이
바늘과 실은 함께 사용 → 세트 관계. ア 열쇠와 자물쇠도 함께 사용 → 세트 ✓ イ 물고기는 바다에 산다 → 역할(장소) ウ 대↔소 → 반의어
정답: ア 鍵 : 錠前 (세트)
Q07
원인과 결과
勉強 : 合格 ア　練習 : 上達　　イ　本 : 図書館 ウ　兄 : 弟　　エ　音楽 : ピアノ
공부 : 합격 과 같은 관계는? ア 연습:상달 イ 책:도서관 ウ 형:동생 エ 음악:피아노
풀이
공부하면 합격한다 → 원인과 결과. ア 연습하면 실력이 오른다 → 원인과 결과 ✓
정답: ア 練習 : 上達 (원인→결과)
Q08
동의어 (어려움)
推量 : 推測 ア　尽力 : 努力　　イ　希望 : 絶望 ウ　帽子 : 頭　　エ　教育 : 学校
추량 : 추측 과 같은 관계는? ア 진력:노력 イ 희망:절망 ウ 모자:머리 エ 교육:학교
풀이
推量=推測 → 동의어. ア 尽力=努力 → 동의어 ✓ イ 희망↔절망 → 반의어
정답: ア 尽力 : 努力 (동의어)
Q09
목적 관계
冷蔵庫 : 保存 ア　時計 : 計測　　イ　映画 : 俳優 ウ　海 : 山　　エ　牛乳 : チーズ
냉장고 : 보존 과 같은 관계는? ア 시계:계측 イ 영화:배우 ウ 바다:산 エ 우유:치즈
풀이
냉장고는 보존을 위해 존재 → 목적 관계. ア 시계는 계측을 위해 존재 → 목적 ✓ イ 배우는 영화에 출연 → 역할 エ 우유로 치즈를 만든다 → 원료
정답: ア 時計 : 計測 (목적)

종합 (포함)
文房具 : ペン ア　乗り物 : 電車　　イ　医者 : 患者 ウ　昼 : 夜　　エ　雨 : 洪水
문방구 : 펜 과 같은 관계는? ア 탈것:전차 イ 의사:환자 ウ 낮:밤 エ 비:홍수
풀이
펜은 문방구의 일종 → 포함(상위:하위). ア 전차는 탈것의 일종 → 포함 ✓ イ 의사와 환자 → 세트/역할 ウ 낮↔밤 → 반의어 エ 비→홍수 → 원인과 결과
정답: ア 乗り物 : 電車 (포함)
연습문제
두 단어의 관계 연습 10문
二語の関係 追加練習10問

기초 4문 → 실전 4문 → 고난도 2문. 제시어 쌍의 관계를 파악한 뒤, 같은 관계의 선택지를 고르세요.

P01 — 동의어 기초
도움 : 원조
助ける : 援助 ア 妨げる : 邪魔　イ 走る : 飛行　ウ 山 : 海 同じ関係のものはどれか。
돕다 : 원조 ア 방해하다 : 방해 イ 달리다 : 비행 ウ 산 : 바다 같은 관계는?
관계
제시: 동의어 (돕다=원조) ア 방해하다=방해 → 동의어 ✓
정답: ア
P02 — 반의어 기초
밝다 : 어둡다
明るい : 暗い ア 高い : 安い　イ 大きい : 巨大　ウ 速い : 走る 同じ関係のものはどれか。
밝다 : 어둡다 ア 비싸다 : 싸다 イ 크다 : 거대 ウ 빠르다 : 달리다 같은 관계는?
관계
제시: 반의어. ア 비싸다↔싸다 → 반의어 ✓ イ 동의어. ウ 형용사:동사(다름)
정답: ア
P03 — 포함 기초
과일 : 사과
果物 : りんご ア 動物 : 犬　イ 花 : 赤い　ウ 本 : 読む 同じ関係のものはどれか。
과일 : 사과 ア 동물 : 개 イ 꽃 : 빨갛다 ウ 책 : 읽다 같은 관계는?
관계
제시: 포함(상위:하위). ア 동물⊃개 → 포함 ✓
정답: ア
P04 — 원료 기초
밀가루 : 빵
小麦粉 : パン ア 木 : 机　イ 水 : 海　ウ 紙 : 白い 同じ関係のものはどれか。
밀가루 : 빵 ア 나무 : 책상 イ 물 : 바다 ウ 종이 : 하얗다 같은 관계는?
관계
제시: 원료→제품. ア 나무→책상(원료) ✓ イ 구성요소이지만 원료가 아님. ウ 명사:형용사
정답: ア
P05 — 역할 관계 실전
의사 : 진찰
医者 : 診察 ア 教師 : 授業　イ 学校 : 勉強　ウ 警察 : 制服 同じ関係のものはどれか。
의사 : 진찰 ア 교사 : 수업 イ 학교 : 공부 ウ 경찰 : 제복 같은 관계는?
관계
제시: 직업:행위(역할). ア 교사→수업 ✓ イ 장소:행위 ウ 직업:소지품
정답: ア
P06 — 세트 관계 실전
바늘 : 실
針 : 糸 ア 鍵 : 錠　イ 山 : 川　ウ 朝 : 夜 同じ関係のものはどれか。
바늘 : 실 ア 열쇠 : 자물쇠 イ 산 : 강 ウ 아침 : 밤 같은 관계는?
관계
제시: 세트(함께 쓰이는 것). ア 열쇠+자물쇠 → 세트 ✓ イ 자연물 나열(세트 아님) ウ 반의어
정답: ア
P07 — 원인·결과 실전
운동 : 건강
運動 : 健康 ア 勉強 : 合格　イ 学校 : 生徒　ウ 病気 : 薬 同じ関係のものはどれか。
운동 : 건강 ア 공부 : 합격 イ 학교 : 학생 ウ 병 : 약 같은 관계는?
관계
제시: 원인→결과. ア 공부→합격 ✓ イ 장소:소속 ウ 문제:해결수단
정답: ア
P08 — 동의어 (난어) 실전
놀이 : 유흥
遊び : 遊興 ア 怒り : 憤慨　イ 喜び : 悲しみ　ウ 寝る : 起きる 同じ関係のものはどれか。
놀이 : 유흥 ア 분노 : 분개 イ 기쁨 : 슬픔 ウ 자다 : 일어나다 같은 관계는?
관계
제시: 동의어(일상어:한자어). ア 분노=분개 → 동의어 ✓ イ 반의어 ウ 반의어
정답: ア
P09 — 복합 판별 고난도
정치 : 국회
政治 : 国会 ア 経済 : 銀行　イ 教育 : 学生　ウ 法律 : 違反 同じ関係のものはどれか。
정치 : 국회 ア 경제 : 은행 イ 교육 : 학생 ウ 법률 : 위반 같은 관계는?
관계
제시: 분야:핵심기관 (정치→국회). ア 경제→은행(핵심기관) ✓ イ 분야:대상자 ウ 규범:위반행위
정답: ア
P10 — 복합 판별 고난도
용기 : 비겁
勇気 : 卑怯 ア 寛容 : 偏狭　イ 正直 : 誠実　ウ 知識 : 学問 同じ関係のものはどれか。
용기 : 비겁 ア 관용 : 편협 イ 정직 : 성실 ウ 지식 : 학문 같은 관계는?
관계
제시: 반의어(긍정↔부정). ア 관용↔편협 → 반의어 ✓ イ 유의어 ウ 포함/관련
정답: ア
