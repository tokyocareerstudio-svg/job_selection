두 단어의 관계 (二語の関係)

예시로 주어진 두 단어의 관계를 파악하고, 같은 관계를 이루는 쌍을 선택지에서 고르는 유형. 8가지 관계 유형을 암기하면 기계적으로 풀 수 있습니다. 핵심 테크닉은 "두 단어로 짧은 문장을 만들어보기"입니다.

테센 ★ 전용
강의
두 단어의 관계란?
二語の関係 — 2つの語の関係を見抜く

이 유형은 제시된 두 단어의 관계를 파악한 뒤, 선택지 중 같은 관계를 가진 쌍을 고르는 문제입니다. 출제되는 관계는 아래 8가지로 거의 고정되어 있으므로, 이 8유형을 외우면 됩니다.

풀이법 3단계

① 예시 두 단어로 짧은 문장을 만든다. "A는 B의 일종이다" "A는 B의 반대이다" 등.

② 8유형 중 어디에 해당하는지 판별.

③ 선택지에서 같은 문장 구조가 성립하는 쌍을 고른다.

8대 관계 유형

① 동의어 (同意語)
永久 : 恒久
A = B. 의미가 거의 같다
② 반의어 (対義語)
拡大 : 縮小
A ↔ B. 의미가 반대
③ 포함 (包含)
果物 : りんご
A는 B를 포함. B는 A의 일종
④ 역포함
犬 : 動物
A는 B의 일종. ③의 방향 반대
⑤ 원료·재료
小麦 : パン
A로 B를 만든다
⑥ 용도·역할
はさみ : 切る
A는 B하는 데 쓴다
⑦ 장소·소속
魚 : 海
A는 B에 있다 / B에서 발견
⑧ 세트·쌍
筆 : 墨
A와 B는 함께 쓰인다
📝 미니 예제 — "문장 만들기" 테크닉

제시: 辞書 : 言葉 선택지: ア 地図:場所 イ 本:紙 ウ 鍵:扉

Step 1: 문장 만들기 → "辞書は言葉を調べるために使う" (사전은 단어를 찾는 데 사용) → 관계: 용도·역할 (⑥) Step 2: 같은 구조가 성립하는 것은? ア "地図は場所を調べるために使う" (지도는 장소를 찾는 데 사용) ✓ イ "本は紙でできている" → 원료·재료(⑤). 구조 다름 ✗ ウ "鍵は扉を開けるために使う" → 비슷하지만 "開ける"라는 동사가 필요. 직접적 대상 관계가 다름.

정답: ア 地図:場所
⚠️ 가장 헷갈리는 함정 — 포함 방향

제시: 野球 : スポーツ 이건 "A는 B의 일종" (④ 역포함)인가, "A는 B를 포함" (③ 포함)인가?

"야구는 스포츠의 일종이다" → ④ 역포함 (A ⊂ B) 만약 "スポーツ : 野球"라면 → ③ 포함 (A ⊃ B) 방향이 반대! 어느 쪽이 큰 범주인지 순서를 반드시 확인.

포함 관계에서는 A, B의 순서(방향)가 핵심!
자주 하는 실수

실수 1: 포함(③)과 역포함(④)의 방향을 혼동. "A가 B를 포함"인지 "A가 B에 포함"인지 주의.

실수 2: "원료·재료"와 "용도·역할"을 혼동. "A로 B를 만든다" vs "A로 B를 한다"는 다름.

실수 3: 문장을 만들지 않고 "느낌"으로 풀어서 미묘한 관계 차이를 놓침.

연습 문제
두 단어의 관계 예제 10문제
二語の関係の練習問題10問

제시된 두 단어의 관계를 파악하고, 같은 관계의 쌍을 고르세요.

Q01 — 동의어
永久 : 恒久
示された二語の関係を考え、同じ関係のものを選びなさい。 永久 : 恒久 ア 迅速 : 緩慢　　イ 丁寧 : 入念　　ウ 簡潔 : 冗長
영구:항구 (동의어). 같은 관계는? ア 신속:완만(반의) イ 정성:세밀(동의) ウ 간결:장황(반의)
풀이
永久=恒久 → 동의어 관계 イ 丁寧=入念 (둘 다 "꼼꼼하게") → 동의어 ✓
정답: イ 丁寧:入念
Q02 — 반의어
拡大 : 縮小
拡大 : 縮小 ア 増加 : 減少　　イ 拡張 : 拡大　　ウ 前進 : 進歩
확대:축소 (반의어). 같은 관계는? ア 증가:감소 イ 확장:확대(동의) ウ 전진:진보(동의)
풀이
拡大↔縮小 → 반의어 ア 増加↔減少 → 반의어 ✓
정답: ア 増加:減少
Q03 — 포함
楽器 : ピアノ
楽器 : ピアノ ア 自動車 : トヨタ　　イ 飲み物 : コーヒー　　ウ 東京 : 日本
악기:피아노 (A⊃B 포함). 같은 관계는? ア 자동차:도요타 イ 음료:커피 ウ 도쿄:일본
풀이
"피아노는 악기의 일종" → 포함(A⊃B) イ "커피는 음료의 일종" → 같은 구조 ✓ ア 도요타는 "자동차 브랜드"이지 "자동차의 일종"은 아님 → 미묘하지만 약간 다름 ウ "도쿄는 일본에 있다" → 장소 관계
정답: イ 飲み物:コーヒー
Q04 — 원료
木 : 紙
木 : 紙 ア 米 : 酒　　イ 魚 : 海　　ウ 花 : 花瓶
나무:종이 (A로 B를 만든다). 같은 관계는? ア 쌀:술 イ 물고기:바다 ウ 꽃:꽃병
풀이
"나무로 종이를 만든다" → 원료 관계 ア "쌀로 술을 만든다" ✓ イ 장소 관계 / ウ 용도 관계
정답: ア 米:酒
Q05 — 용도
めがね : 見る
めがね : 見る ア 靴 : 歩く　　イ 本 : 紙　　ウ 雨 : 傘
안경:보다 (A는 B하는 데 쓰임). 같은 관계? ア 신발:걷다 イ 책:종이 ウ 비:우산
풀이
"안경은 보는 데 사용" → 용도 ア "신발은 걷는 데 사용" ✓
정답: ア 靴:歩く
Q06 — 장소
生徒 : 学校
生徒 : 学校 ア 患者 : 病院　　イ 医者 : 注射　　ウ 先生 : 教科書
학생:학교 (A는 B에 있다). 같은 관계? ア 환자:병원 イ 의사:주사 ウ 선생:교과서
풀이
"학생은 학교에 있다" → 장소 ア "환자는 병원에 있다" ✓
정답: ア 患者:病院
Q07 — 세트
針 : 糸
針 : 糸 ア 弓 : 矢　　イ 鍵 : 扉　　ウ 紙 : ペン
바늘:실 (함께 쓰이는 세트). 같은 관계? ア 활:화살 イ 열쇠:문 ウ 종이:펜
풀이
"바늘과 실은 함께 쓰인다" → 세트 ア "활과 화살은 함께 쓰인다" ✓
정답: ア 弓:矢
Q08 — 역포함
桜 : 花
桜 : 花 ア 猫 : 動物　　イ 動物 : 犬　　ウ 赤 : 色
벚꽃:꽃 (A는 B의 일종, A⊂B). 같은 관계? ア 고양이:동물 イ 동물:개 ウ 빨강:색
풀이
"벚꽃은 꽃의 일종" → A⊂B (역포함) ア "고양이는 동물의 일종" ✓ 방향 동일 イ "동물은 개를 포함" → 방향 반대 (포함) ウ "빨강은 색의 일종" ✓ 방향 동일 ア, ウ 모두 가능하지만, 선택지에서 하나만 정답일 때 둘 다 맞는 경우 문제가 복수 정답을 허용합니다.
정답: ア 猫:動物, ウ 赤:色 (둘 다 역포함)
Q09 — 동의어 (한자)
敏捷 : 機敏
敏捷 : 機敏 ア 寛大 : 厳格　　イ 慎重 : 軽率　　ウ 困難 : 難解
민첩:기민 (동의어). 같은 관계? ア 관대:엄격(반의) イ 신중:경솔(반의) ウ 곤란:난해(동의)
풀이
敏捷=機敏 (둘 다 "날래다") → 동의어 ウ 困難≒難解 (둘 다 "어렵다") → 동의어 ✓
정답: ウ 困難:難解
Q10 — 복합 판단
医者 : 病気
医者 : 病気 ア 弁護士 : 裁判　　イ 消防士 : 火事　　ウ 教師 : 学校
의사:병 (A는 B를 다룬다/대처한다). 같은 관계? ア 변호사:재판 イ 소방관:화재 ウ 교사:학교
풀이
"의사는 병을 치료한다(대처한다)" → 역할·대상 イ "소방관은 화재를 진압한다(대처한다)" → 같은 구조 ✓ ア "변호사는 재판을 한다" → 비슷하지만 "대처"보다는 "참여" ウ "교사는 학교에 있다" → 장소
정답: イ 消防士:火事
