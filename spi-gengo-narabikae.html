문장 배열·빈칸 채우기

흩어진 문장이나 문절을 논리적 순서로 재배열하거나, 빈칸에 적절한 접속사·표현을 넣는 유형. 테센과 웹테 모두 출제됩니다. "접속사를 단서로 논리 흐름을 읽는 것"이 핵심입니다.

공통 출제
강의
문장 배열·빈칸의 핵심 전략
文の並べ替え・空欄補充 — 接続詞がカギ

이 유형은 논리적 사고력과 일본어 문장 이해력을 동시에 측정합니다. 핵심은 접속사와 지시어(それ·これ 등)를 단서로 문장의 전후 관계를 파악하는 것입니다.

풀이법

① 접속사 단서 찾기. しかし(역접)→앞과 반대, したがって(순접)→앞의 결론, また(병렬)→같은 레벨 추가, たとえば(예시)→앞의 구체화

② 지시어 추적. "それ"가 있으면 바로 앞에 그것이 가리키는 내용이 있어야 함. "このように"가 있으면 앞에 구체적 내용이 있어야 함.

③ 확정 쌍 먼저. 확실히 연결되는 2문장을 먼저 묶고, 나머지를 채운다.

④ 빈칸 채우기: 빈칸 앞뒤의 논리 관계(순접/역접/추가/예시)를 파악하고, 그에 맞는 접속사를 넣는다.

빈출 접속사 정리

순접 (따라서): したがって, そのため, だから, ゆえに
역접 (그러나): しかし, だが, ところが, にもかかわらず
추가 (또한): また, さらに, その上, 加えて
예시 (예를 들어): たとえば, 具体的には, いわば
전환 (한편): 一方, 他方, これに対して
정리 (즉): つまり, すなわち, 要するに

📝 미니 예제 — 빈칸 채우기

日本は資源が少ない。（　　）、技術力で経済を発展させてきた。 ア しかし イ つまり ウ たとえば

"자원이 적다" → (역접) → "기술력으로 발전시켰다" 자원 부족에도 불구하고 발전 → しかし (그러나)

정답: ア しかし

並べ替え・空欄補充の練習問題10問
Q01 — 빈칸 (역접)
접속사 선택
彼は努力した。（　　）、結果は出なかった。 ア したがって　イ しかし　ウ つまり
그는 노력했다. ( ) 결과는 나오지 않았다.
노력(+) → 결과 안 나옴(-) = 역접
정답: イ しかし
Q02 — 빈칸 (순접)
접속사 선택
実験は成功した。（　　）、この理論の正しさが証明された。 ア しかし　イ したがって　ウ 一方
실험은 성공했다. ( ) 이 이론의 정당성이 증명되었다.
실험 성공 → 이론 증명 = 순접 (원인→결과)
정답: イ したがって
Q03 — 빈칸 (예시)
접속사 선택
日本には多くの伝統行事がある。（　　）、正月には初詣に行く習慣がある。 ア しかし　イ つまり　ウ たとえば
일본에는 많은 전통 행사가 있다. ( ) 정월에는 하쓰모데에 가는 습관이 있다.
전통 행사(일반) → 하쓰모데(구체적 예) = 예시
정답: ウ たとえば
Q04 — 빈칸 (정리)
접속사 선택
運動をすれば体力がつく。睡眠を十分にとれば疲れが取れる。（　　）、健康には生活習慣が大切なのだ。 ア たとえば　イ しかし　ウ つまり
운동하면 체력이 붙는다. 충분히 자면 피로가 풀린다. ( ) 건강에는 생활 습관이 중요하다.
구체적 내용 → 요약/정리 = つまり
정답: ウ つまり
Q05 — 빈칸 (추가)
접속사 선택
この商品は品質が良い。（　　）、価格も手頃である。 ア しかし　イ さらに　ウ つまり
이 상품은 품질이 좋다. ( ) 가격도 적당하다.
장점1 → 장점2 추가 = さらに
정답: イ さらに
Q06 — 문장 배열
3문장 배열
次のア～ウを意味が通るように並べ替えなさい。 ア しかし、最近は減少傾向にある。 イ 日本の人口は長年増加してきた。 ウ その原因は少子化だと言われている。
ア 그러나 최근에는 감소 추세. イ 일본 인구는 오랜 기간 증가. ウ 그 원인은 저출산이라고 한다.
イ(증가) → ア(しかし=역접, 감소) → ウ(その原因=앞 내용의 원인)
정답: イ → ア → ウ
Q07 — 문장 배열
4문장 배열
ア 具体的には、週3回の運動が推奨されている。 イ 健康を維持するには運動が重要だ。 ウ このように、適度な運動は心身に良い影響を与える。 エ それにより、ストレスの軽減や体力の向上が期待できる。
ア 구체적으로 주 3회 운동 추천. イ 건강 유지에 운동 중요. ウ 이와 같이, 적당한 운동은 심신에 좋은 영향. エ 그에 따라 스트레스 감소와 체력 향상 기대.
イ(주장) → ア(具体的には=구체화) → エ(それにより=결과) → ウ(このように=정리)
정답: イ → ア → エ → ウ
Q08 — 빈칸 (전환)
접속사 선택
A社は国内市場を重視している。（　　）、B社は海外展開に力を入れている。 ア つまり　イ 一方　ウ したがって
A사는 국내 시장 중시. ( ) B사는 해외 전개에 주력.
A사(국내) vs B사(해외) = 대조·전환
정답: イ 一方
Q09 — 빈칸 (역접)
접속사 선택
この薬は効果がある。（　　）、副作用もある。 ア したがって　イ たとえば　ウ ただし
이 약은 효과가 있다. ( ) 부작용도 있다.
효과(+) → 부작용(-) = 역접/단서 → ただし(단,)
정답: ウ ただし
Q10 — 문장 배열
3문장 배열
ア だから、早めの対策が必要だ。 イ 地球温暖化は年々深刻化している。 ウ たとえば、再生可能エネルギーの活用が挙げられる。
ア 그래서 조기 대책 필요. イ 지구온난화는 해마다 심각화. ウ 예를 들어 재생에너지 활용.
イ(현상) → ア(だから=순접, 대책 필요) → ウ(たとえば=대책의 구체적 예)
정답: イ → ア → ウ
