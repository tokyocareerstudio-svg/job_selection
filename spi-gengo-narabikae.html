문장 배열 (文の並び替え)

문장 조각들을 논리적 순서로 배열하거나, 빈칸에 적절한 접속사를 넣는 유형. 테스트센터와 웹테스팅 모두 출제. 접속사의 종류와 역할을 암기하면 빠르게 풀 수 있습니다.

테센 + 웹 공통 출제
강의
문장 배열·빈칸 채우기란?
文の並び替え·空欄補充 — 接続詞と指示語がカギ

이 유형은 두 가지 형태로 출제됩니다:

① 문장 배열 (文の並び替え): 섞여 있는 4~5개의 문장 조각을 올바른 순서로 배열하는 문제.
② 빈칸 채우기 (空欄補充): 문장의 빈칸에 들어갈 적절한 접속사나 어구를 선택하는 문제.

테스트센터·웹테스팅 양쪽 모두 출제됩니다. 두 유형 모두 접속사(接続詞)의 역할을 이해하는 것이 핵심입니다.

풀이 핵심 4단계

1단계: 접속사를 먼저 찾아라. しかし(그러나), したがって(따라서), また(또한) 등이 논리 관계의 단서.

2단계: 지시어를 추적하라. "その(그)", "このような(이러한)" 등이 가리키는 내용이 반드시 앞에 나와야 함.

3단계: 첫 문장과 마지막 문장을 먼저 결정. 도입부(배경·문제제기)와 결론부(주장·요약)를 찾으면 중간 배열이 쉬워짐.

4단계: 논리 흐름 확인. 일본어 문장은 보통 "현상→원인→대책→결과" 또는 "주장→근거→예시→결론" 순서.

접속사별 풀이 — 이것만 알면 빈칸 채우기는 해결

접속사는 앞뒤 문장의 논리 관계를 결정합니다. 앞 문장과 뒷 문장의 내용이 같은 방향인지, 반대 방향인지를 파악하면 접속사가 결정됩니다.

📝 미니 예제 — 역접 (반전)

日本経済は回復傾向にある。（　）、失業率はまだ高い水準にある。 ア　したがって　　イ　しかし　　ウ　つまり　　エ　たとえば

풀이: 앞: "경제 회복 중" (긍정) → 뒤: "실업률 아직 높음" (부정) 방향이 반대 → 역접 접속사가 필요 → イ しかし(그러나) ✓

정답: イ しかし
📝 미니 예제 — 순접 (원인→결과)

彼は毎日練習を欠かさない。（　）、大会では常に上位入賞する。 ア　しかし　　イ　そのため　　ウ　ところで　　エ　一方

풀이: 앞: "매일 연습" (원인) → 뒤: "대회 상위 입상" (결과) 원인→결과의 순접 → イ そのため(그 때문에) ✓

정답: イ そのため

문장 배열 — 지시어가 열쇠

문장 배열에서는 지시어(この·その·こうした 등)가 가리키는 내용이 반드시 앞에 나와야 합니다. 이 원칙 하나만으로 순서를 크게 좁힐 수 있습니다.

📝 미니 예제 — 문장 배열

ア　そのため、多くの企業が対策を講じている。 イ　近年、サイバー攻撃が急増している。 ウ　具体的には、セキュリティ研修の義務化などが挙げられる。 エ　被害額は年間数兆円に上るとされる。

풀이: • イ "근년 사이버 공격 급증" → 도입부(현상 제시) → 첫 문장 • エ "피해액 수조 엔" → イ의 구체적 규모 → 2번째 • ア "そのため(그 때문에) 기업이 대책" → エ의 결과 → 3번째 • ウ "具体的には(구체적으로)" → ア의 대책 예시 → 마지막 지시어 추적: そのため→앞에 원인 필요 / 具体的には→앞에 총론 필요

정답: イ → エ → ア → ウ
⚠️ 함정 — 確かに~しかし 패턴

빈칸 채우기에서 "確かに~" 뒤에 "しかし"가 오는 패턴이 자주 나옵니다. "確かに(확실히)"는 상대방 의견을 양보하는 것이고, 진짜 주장은 "しかし" 뒤에 나옵니다. 이 패턴을 모르면 양보 부분을 주장으로 오해합니다.

예시: "確かに、AIは多くの仕事を奪うだろう。しかし、（　）" → しかし 뒤에는 앞과 반대 내용이 와야 함 → "인간만의 창의성은 대체 불가" 같은 내용이 정답

確かに = 양보, しかし 뒤 = 진짜 주장
시간 관리

빈칸 채우기: 1문제 30~40초. 앞뒤 문장의 방향(긍정/부정, 원인/결과)만 파악하면 접속사가 결정됨.
문장 배열: 1문제 50~60초. 접속사와 지시어를 먼저 찾고, 첫/마지막 문장부터 결정하세요.

연습 문제
문장 배열·빈칸 채우기 예제 10문제
文の並び替え·空欄補充の練習問題10問

접속사와 지시어를 단서로 논리적 순서를 파악하세요!

Q01
접속사 선택
次の文の（　）に入る最も適切なものを選びなさい。 日本経済は回復傾向にある。（　）、失業率はまだ高い水準にある。 ア　したがって イ　しかし ウ　つまり エ　たとえば
"일본 경제는 회복 추세에 있다. ( ) 실업률은 아직 높은 수준이다." ア 따라서 イ 그러나 ウ 즉 エ 예를 들어
풀이
앞: 경제 회복 (긍정) → 뒤: 실업률 높음 (부정) → 역접이 필요 イ しかし(그러나) = 역접 접속사 ✓
정답: イ しかし
Q02
접속사 선택
彼は毎日練習を欠かさない。（　）、大会では常に上位入賞する。 ア　しかし イ　そのため ウ　ところで エ　一方
"그는 매일 연습을 빠뜨리지 않는다. ( ) 대회에서는 항상 상위 입상한다." ア 그러나 イ 그 때문에 ウ 그런데 エ 한편
풀이
매일 연습(원인) → 상위 입상(결과) → 순접이 필요 イ そのため(그 때문에) = 순접 ✓
정답: イ そのため
Q03
접속사 선택
この地域は夏に雨が多い。（　）、冬はほとんど雨が降らない。 ア　また イ　したがって ウ　一方 エ　つまり
"이 지역은 여름에 비가 많다. ( ) 겨울에는 거의 비가 오지 않는다." ア 또한 イ 따라서 ウ 한편 エ 즉
풀이
여름(비 많음) vs 겨울(비 적음) → 대조적 내용 → 전환·대비 ウ 一方(한편) = 대비·전환 ✓
정답: ウ 一方
Q04
문장 배열
次のア~エを意味が通るように並べ替えなさい。 ア　そのため、多くの企業が対策を講じている。 イ　近年、サイバー攻撃が急増している。 ウ　具体的には、セキュリティ研修の義務化などが挙げられる。 エ　被害額は年間数兆円に上るとされる。
ア~エを 올바른 순서로 배열하세요. ア 그 때문에 많은 기업이 대책을 강구하고 있다. イ 근년 사이버 공격이 급증하고 있다. ウ 구체적으로는 보안 연수 의무화 등이 있다. エ 피해액은 연간 수조 엔에 달한다고 한다.
풀이
イ(현상 제시) → エ(피해 규모) → ア(そのため=순접 → 대책) → ウ(具体的には=예시) 논리 흐름: 현상→규모→대책→구체적 예시
정답: イ → エ → ア → ウ
Q05
접속사 선택
日本語は主語を省略することが多い。（　）、英語では主語を必ず明示する。 ア　同様に イ　それに対して ウ　したがって エ　すなわち
"일본어는 주어를 생략하는 경우가 많다. ( ) 영어에서는 주어를 반드시 명시한다." ア 마찬가지로 イ 그에 반해 ウ 따라서 エ 즉
풀이
일본어(생략) vs 영어(명시) → 대비 イ それに対して(그에 반해) = 대비 ✓
정답: イ それに対して
Q06
문장 배열
ア　しかし、最近では在宅勤務が普及し始めた。 イ　かつて日本では、毎日オフィスに出勤するのが当然だった。 ウ　その結果、働き方の選択肢が広がっている。 エ　コロナ禍がその変化を加速させた。
올바른 순서로 배열하세요. イ 과거 일본에서는 매일 출근이 당연했다. ア 그러나 최근에는 재택근무가 보급되기 시작했다. エ 코로나 사태가 그 변화를 가속시켰다. ウ 그 결과 일하는 방식의 선택지가 넓어지고 있다.
풀이
イ(과거) → ア(しかし=반전, 최근 변화) → エ(가속 요인) → ウ(その結果=결과) 시간순: 과거→반전→가속→결과
정답: イ → ア → エ → ウ
Q07
접속사 선택
この計画にはリスクがある。（　）、実行する価値は十分にある。 ア　だから イ　それでも ウ　ところで エ　つまり
"이 계획에는 리스크가 있다. ( ) 실행할 가치는 충분히 있다." ア 그래서 イ 그럼에도 ウ 그런데 エ 즉
풀이
리스크(부정) → 그래도 가치 있음(긍정) → 역접 イ それでも(그럼에도) = 역접 ✓
정답: イ それでも
Q08
접속사 선택
彼女は英語が堪能だ。（　）、フランス語も話せる。 ア　しかし イ　さらに ウ　つまり エ　そこで
"그녀는 영어에 능통하다. ( ) 프랑스어도 할 수 있다." ア 그러나 イ 게다가 ウ 즉 エ 거기서
풀이
영어 능통 + 프랑스어도 → 추가·첨가 イ さらに(게다가) = 첨가 ✓
정답: イ さらに
Q09
문장 배열
ア　たとえば、運動不足や睡眠の質の低下が指摘されている。 イ　現代人の健康問題が深刻化している。 ウ　こうした問題に対して、企業も健康経営に取り組み始めた。 エ　特にデスクワーク中心の働き方が影響していると言われる。
올바른 순서로 배열하세요.
풀이
イ(총론) → エ(특히=구체화) → ア(たとえば=예시) → ウ(こうした=앞 내용 받아 대책) 흐름: 문제 제기→원인→예시→대책
정답: イ → エ → ア → ウ
Q10
접속사 선택
売上は前年比10%増加した。（　）、利益率は逆に低下した。 ア　そのため イ　ところが ウ　また エ　すなわち
"매출은 전년 대비 10% 증가했다. ( ) 이익률은 오히려 저하했다." ア 그 때문에 イ 그런데 ウ 또한 エ 즉
풀이
매출 증가(긍정) → 이익률 저하(예상 밖 부정) → 역접(예상 밖) イ ところが = 예상 밖의 결과를 나타내는 역접 ✓
정답: イ ところが
