SECTION 01
TG-WEB이란?
TG-WEBとは — ヒューマネージ社の高難度適性検査

TG-WEB은 株式会社ヒューマネージ가 개발·판매하는 적성검사로, SPI·玉手箱와 함께 일본 취활 3대 웹테스트 중 하나입니다. 2,000사 이상이 도입하고 있으며, 자생당(資生堂), 기린(キリン), 산토리(サントリー), 미쓰비시지쇼(三菱地所), TV아사히 등 대기업에서 많이 채용하고 있습니다.

SPI · 玉手箱와의 핵심 차이

난이도가 압도적으로 높습니다. SPI가 「기초학력 측정」, 玉手箱가 「스피드 처리력 측정」이라면, TG-WEB은 「응용사고력 측정」입니다. 특히 종래형 계수에서는 전개도, 암호해독, 거짓말쟁이 추론 등 SPI/玉手箱에서는 절대 나오지 않는 유형이 출제됩니다.

반면 보더라인(합격선)은 4-5할(40-50%)로 SPI(7할)보다 낮습니다. 어려운 만큼 만점을 기대하지 않으며, 「풀 수 있는 문제를 확실히 맞추는 전략」이 핵심입니다.

TG-WEB 판별법 — URL로 확인

기업에서 보내주는 수험 URL에 c-personal.com 또는 e-gitest.com이 포함되어 있으면 TG-WEB입니다. SPI(arorua.net)나 玉手箱(e-exams.jp)와는 URL이 다르므로, 수험 전에 어떤 시험인지 확인할 수 있습니다.

종래형 vs 신형 — 완전히 다른 시험

TG-WEB의 가장 큰 특징은 언어와 계수에 「종래형(구형)」과 「신형(신형)」 2종류가 존재한다는 것입니다. 기업마다 어느 쪽을 채용하는지가 다르며, 대다수 기업은 종래형을 사용합니다. 시험 시작 시 제한시간으로 판별할 수 있습니다.

종래형 vs 신형 판별법

시험 시작 전 과목 일람 화면에서 제한시간을 확인하세요.
• 검사1 = 12분 → 종래형 언어 / 검사2 = 18분 → 종래형 계수
• 검사1 = 7분 → 신형 언어 / 검사2 = 8분 → 신형 계수

대부분의 기업은 언어→계수→성격검사 순서로 진행됩니다. 시험 시간만 보면 어떤 형식인지 즉시 알 수 있습니다.

SPI / 玉手箱 대비와의 관계

• 신형 계수 = 사칙역산 + 도표읽기 → 玉手箱 계수 대비가 그대로 적용
• 신형 언어 = 동의어/반의어/속담 → SPI 언어 어휘 대비 + 속담 암기 추가
• 종래형 계수 = 전개도/추론/암호 → TG-WEB 전용 대비 필수 (SPI/玉手箱 경험이 전혀 도움 안 됨)
• 종래형 언어 = 논리 독해 → 대학입시 현대문 수준, SPI보다 어려움

수험 형식 비교
受験形式の比較 — WEB vs テストセンター vs マークシート

WEB 방식 (자택 수험) — 가장 흔함

• 기업이 지정한 URL에 자택 PC로 접속하여 수험
• 전자계산기 사용 가능 — 계수 풀 때 계산기를 미리 준비
• SPI와 달리 문제 간 자유 이동 가능 — 어려운 문제를 건너뛰고 쉬운 것부터 풀 수 있음
• TG-WEB eye(AI 감시형)가 도입된 기업도 있음 — 카메라 ON 요구 시 부정행위 탐지

테스트센터 방식 (회장 수험)

• 전국 주요 도시의 테스트센터에서 PC로 수험
• 전자계산기 사용 불가 — 암산·필산 필수, 메모용지 제공
• 계산 속도가 결과에 직결되므로, 테스트센터 수험이 예상되면 필산 연습 병행

마크시트 방식 (기업 회장)

• 기업 회장에서 필기시험으로 실시
• WEB 방식과 동일한 문제가 출제되는 경우가 많음

SECTION 04
학습 순서 추천
おすすめの学習順序
추천 학습 로드맵

STEP 1. 종래형 계수 — 전개도·추론 패턴 암기 (최우선)
TG-WEB에서 가장 어렵고, SPI/玉手箱 경험이 전혀 통하지 않는 영역. 정육면체·정팔면체 전개도 11종 암기와 거짓말쟁이 추론 패턴을 먼저 잡으세요.

STEP 2. 종래형 언어 — 공란보충·나열 연습
접속사·지시어 분석으로 패턴화 가능. 공란+나열 6문만 맞추면 50%에 도달.

STEP 3. 신형 계수/언어 — 玉手箱·SPI 대비로 대체
신형 계수는 玉手箱 사칙역산·도표읽기와 동일, 신형 언어는 SPI 어휘 대비 + 속담 암기. 이미 玉手箱·SPI를 공부했다면 추가 대비 최소화 가능.

STEP 4. 시간 관리 연습
종래형은 시작 시 전 문제를 스캔하고 쉬운 것부터 풀기. 신형은 1문 5-13초 페이스를 체감하기. 반드시 타이머를 켜고 실전과 동일한 조건에서 연습.

SPI·玉手箱와 병행 학습 시 우선순위

대부분의 취활생은 SPI, 玉手箱, TG-WEB을 모두 대비해야 합니다. 시간이 한정되어 있으므로 우선순위가 중요합니다.

• SPI 먼저 — 도입률 최고, 기초가 되는 시험. SPI 대비가 신형 TG-WEB에도 적용됨
• 玉手箱 다음 — 점유율 1위 웹테스트. 玉手箱 계수 대비가 신형 TG-WEB 계수에 그대로 적용
• TG-WEB 종래형은 마지막 — 도입 기업이 SPI/玉手箱보다 적으므로, 지원 기업이 TG-WEB을 채용하는 것이 확인된 후 집중 대비
