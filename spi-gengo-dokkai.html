장문 독해 (長文読解)

400~600자 분량의 지문을 읽고 내용 이해·요지 파악·추론 문제를 푸는 유형. 테스트센터와 웹테스팅 모두 출제. 배점이 높고 시간 소모가 크므로, 앞의 어휘 문제를 빨리 풀고 여기에 시간을 남기는 것이 전략입니다.

장문 독해란?
長文読解 — 時間配分が最重要

SPI 언어 파트에서 가장 배점이 높고 가장 시간이 오래 걸리는 유형입니다. 400~600자 분량의 일본어 지문이 주어지고, 그에 대해 3~4개의 질문이 나옵니다. 테스트센터·웹테스팅 양쪽 모두 출제.

어구의 의미(20초), 두 단어 관계(10~20초) 등 암기형 문제를 빠르게 끝내고 장문 독해에 충분한 시간을 확보하는 것이 SPI 언어 전체의 핵심 전략입니다.

장문 독해 4단계 풀이법

1단계: 질문을 먼저 읽어라. 지문을 읽기 전에 질문을 확인하면 "무엇을 찾아야 하는지" 초점이 생김. 이것만으로 읽기 속도가 체감 2배.

2단계: 주장 문장을 찾아라. 필자의 핵심 주장은 보통 마지막 단락이나 접속사 "つまり(즉)" 뒤에 있음.

3단계: 접속사로 단락 구조를 파악하라. しかし(반전), つまり(요약), たとえば(예시) 등이 각 단락의 역할을 알려줌.

4단계: 소거법으로 선택지를 줄여라. 명백히 틀린 선택지를 먼저 제거하면 정답률이 크게 올라감.

질문 유형 3가지

유형 A: 내용 일치 (内容一致) — 가장 빈출

"본문의 내용과 일치하는 것을 고르시오" 형태. 지문의 각 단락을 정확히 읽으면 풀 수 있습니다.

📝 미니 예제 — 내용 일치

「日本の人口は2008年をピークに減少に転じた。特に地方では過疎化が進み、多くの自治体で存続が危ぶまれている。一方、東京圏への人口集中は依然として続いている。」 ア　日本の人口は現在も増加している イ　地方の過疎化は改善傾向にある ウ　東京圏への人口集中が続いている

풀이: 지문에서 직접 확인: ア → "減少に転じた"이므로 증가는 ✗ イ → "過疎化が進み"이므로 개선은 ✗ ウ → "依然として続いている" = 여전히 계속 → 일치 ✓

정답: ウ

내용 일치 문제에서 자주 나오는 4가지 함정 패턴을 알아두세요:

내용 일치 4대 함정

① 범위 확대: "일부"를 "전부"로 바꿔서 틀린 선택지. 全て·すべて·必ず 등 절대적 표현 주의.

② 인과 뒤바꿈: 원인과 결과를 반대로 연결. "A때문에 B" → 선택지에서 "B때문에 A"로 변형.

③ 부분 인용: 지문의 일부만 인용해서 전체 의미를 왜곡. 해당 부분의 전후 문맥을 확인.

④ 유사 표현 치환: 비슷하지만 의미가 다른 표현으로 교체. 원문 키워드를 정확히 기억.

유형 B: 요지 파악 (要旨把握)

"필자의 주장으로 가장 적절한 것을 고르시오" 형태. 지문 전체를 관통하는 핵심 메시지를 파악해야 합니다.

주장이 나오는 시그널

직접적 주장: 重要なのは(중요한 것은), ~べきである(~해야 한다), ~ではないだろうか(~가 아닐까)

요약 시그널: つまり(즉), すなわち(다시 말해), 結論として(결론적으로)

주의: 確かに~しかし 패턴에서 確かに 뒤는 양보(상대방 의견), しかし 뒤가 진짜 주장!

📝 미니 예제 — 요지 파악

「多くの人は失敗を恐れて挑戦を避ける。確かに、失敗には痛みが伴う。しかし、失敗から学ぶことこそが成長の最大のエンジンである。」 筆者の主張はどれか。 ア　失敗は避けるべきだ イ　失敗から学ぶことが成長につながる ウ　挑戦しないことが最善だ

풀이: 確かに(양보) → "실패에는 고통이 따른다" しかし(반전) → "실패에서 배우는 것이야말로 성장의 최대 엔진" → しかし 뒤가 진짜 주장 = イ ✓ ア는 確かに 부분(양보)을 주장으로 오해한 함정.

정답: イ

유형 C: 공란 보충 (空欄補充)

지문 중간의 빈칸에 들어갈 적절한 어구나 문장을 고르는 유형. 빈칸의 앞뒤 문맥, 특히 접속사와 지시어를 확인하면 정답을 좁힐 수 있습니다.

📝 미니 예제 — 공란 보충

「読書の価値は、単に知識を得ることだけではない。他者の経験や感情を追体験することで、（　　）が養われる。」 ア　計算力　　イ　共感力　　ウ　体力　　エ　記憶力

풀이: "타인의 경험과 감정을 추체험" → 남의 감정을 이해하는 능력 = イ 共感力(공감력) ✓ 문맥 키워드: "他者の感情" → "共感"

정답: イ 共感力

Q01
내용 일치
次の文を読んで、内容と一致するものを選びなさい。 「日本の人口は2008年をピークに減少に転じた。特に地方では過疎化が進み、多くの自治体で存続が危ぶまれている。一方、東京圏への人口集中は依然として続いており、地方と都市の格差は拡大している。」 ア　日本の人口は現在も増加している イ　地方の過疎化は改善傾向にある ウ　東京圏への人口集中が続いている エ　地方と都市の格差は縮小している
"일본 인구는 2008년을 정점으로 감소로 전환했다. 특히 지방에서는 과소화가 진행되어... 한편 도쿄권 인구 집중은 여전히 계속..." 내용과 일치하는 것은?
풀이
지문: "東京圏への人口集中は依然として続いており" → ウ "도쿄권 인구 집중이 계속되고 있다" = 일치 ✓ ア 인구 감소(증가X) / イ 과소화 진행 중(개선X) / エ 격차 확대(축소X)
정답: ウ
Q02
요지 파악
「AIの発展により、多くの職業が自動化される可能性がある。しかし、創造性や共感力を必要とする仕事は簡単には代替されないだろう。重要なのは、AIを恐れるのではなく、AIと共存するためのスキルを身につけることである。」 筆者の主張として最も適切なものはどれか。 ア　AIにより全ての仕事がなくなる イ　AIと共存するスキルを身につけるべきだ ウ　創造的な仕事だけが残る エ　AIの発展を止めるべきだ
필자의 주장으로 가장 적절한 것은?
풀이
주장 시그널: "重要なのは(중요한 것은)" → "AIと共存するためのスキルを身につけること" = イ "AI와 공존하는 스킬을 익혀야 한다" ✓
정답: イ
Q03
내용 일치
「日本の四季は、世界的に見ても特徴的である。春の桜、夏の花火、秋の紅葉、冬の雪景色と、季節ごとに異なる美しさがある。ただし、近年は地球温暖化の影響で、桜の開花時期が早まるなど、従来の季節感が変化しつつある。」 ア　日本の四季は世界中どこでも同じだ イ　桜の開花時期に変化は見られない ウ　地球温暖化が日本の季節感に影響を与えている エ　冬の雪景色は年々増えている
내용과 일치하는 것은?
풀이
지문: "地球温暖化の影響で...従来の季節感が変化しつつある" → ウ "지구온난화가 일본의 계절감에 영향" = 일치 ✓
정답: ウ

"독서의 가치는 단순히 지식을 얻는 것만이 아니다. 타인의 경험과 감정을 추체험함으로써 ( )가 길러진다." 빈칸에 들어갈 것은?

Q05
요지 파악
「終身雇用制度は長らく日本企業の特徴であった。しかし、グローバル化やIT革命により、企業は柔軟な人材活用を求めるようになった。もはや一つの会社に定年まで勤めることが当然ではない時代に、個人もまたキャリアを主体的に設計する必要がある。」 筆者の主張はどれか。 ア　終身雇用を維持すべきだ イ　個人が主体的にキャリアを設計すべきだ ウ　グローバル化は日本に悪影響を与えた エ　IT革命により仕事がなくなった
필자의 주장은?
풀이
주장 시그널: "個人もまた...必要がある" → 개인이 주체적으로 커리어를 설계해야 한다 = イ ✓
정답: イ

내용 일치

내용과 일치하는 것은?
풀이
지문: "生活インフラとしての役割を果たしている" → イ "편의점은 생활 인프라 역할을 한다" = 일치 ✓

빈칸에 들어갈 것은? "어떤 전문적 내용이라도 상대에게 ( )면 의미가 없다"

Q09
내용 일치
「リモートワークの普及により、都心のオフィス空室率が上昇した。一部の企業はオフィスを縮小し、その分を従業員の在宅勤務手当に充てている。ただし、対面でのコミュニケーションの重要性を再認識し、週に数日は出社を求める企業も増えている。」 ア　全ての企業がオフィスを廃止した イ　対面コミュニケーションは不要になった ウ　一部企業は出社と在宅を組み合わせている エ　オフィス空室率は低下している
내용과 일치하는 것은?
풀이
"週に数日は出社を求める企業も増えている" → ウ "일부 기업은 출근과 재택을 조합하고 있다" = 일치 ✓ ア 전부X / イ 불필요X(재인식) / エ 상승(저하X)
정답: ウ
Q10
요지 파악
「外国語学習において、文法の正確さにこだわりすぎると、会話の流暢さが損なわれる。逆に、文法を無視すれば、正確な意思疎通が困難になる。つまり、文法と実践のバランスこそが、効果的な外国語習得の鍵なのである。」 筆者の主張はどれか。 ア　文法だけを学ぶべきだ イ　会話練習だけが重要だ ウ　文法と実践のバランスが重要だ エ　外国語学習は無意味だ
필자의 주장은?
풀이
주장 시그널: "つまり(즉)" → "文法と実践のバランスこそが...鍵" = ウ "문법과 실천의 밸런스가 중요하다" ✓
정답: ウ
