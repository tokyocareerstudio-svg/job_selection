논리적독해 (論理的読解)

약 600자의 장문을 읽고, 설문문이 논리적으로 A(정확히 맞다) B(정확히 틀리다) C(판단할 수 없다)인지 판별하는 유형. GAB 페이퍼 시험에서 유래한 형식으로, 「본문에 쓰여 있는 것만으로 판단」하는 것이 철칙입니다. 상식이나 추측을 섞는 순간 오답.

32문 (8장문 x 4문) / 15분
강의
GAB형 논리적독해란?
GAB形式の論理的読解 — 本文だけで判断する

玉手箱 언어에서 가장 자주 출제되는 형식입니다. 약 600자의 일본어 장문이 주어지고, 그에 대한 설문문 4개가 표시됩니다. 각 설문문에 대해 아래 3택 중 하나를 고릅니다.

3택 선택지 (전 문제 공통)

A. 본문에서 논리적으로 생각하여, 설문문은 분명히 맞다
→ 본문에 직접적 근거가 있고, 그로부터 논리적으로 도출 가능

B. 본문에서 논리적으로 생각하여, 설문문은 분명히 틀리다
→ 본문의 내용과 명백히 모순되거나 반대

C. 본문만으로는 설문문이 맞는지 틀리는지 판단할 수 없다
→ 본문에 관련 정보가 없거나, 맞을 수도 틀릴 수도 있는 경우

가장 중요한 원칙은 「본문에 쓰여 있는 것만으로 판단」하는 것입니다. 본문에 없는 상식, 일반론, 개인적 의견을 섞어서는 안 됩니다. 아무리 세상에서 사실이어도 본문에 언급이 없으면 C입니다.

출제 형태 — 이렇게 나옵니다

화면 상단에 장문이 표시되고, 그 아래에 설문문 4개가 나옵니다. 각각에 A/B/C를 선택합니다. 4문 모두 응답하고 '다음'을 누르면 다음 장문으로 넘어갑니다. 한번 넘기면 돌아올 수 없으니 주의하세요.

📝 미니 예제

【본문 (발췌)】 「近年、リモートワークの普及により、都心部のオフィス需要は減少傾向にある。一方で、地方都市では移住者の増加に伴い、コワーキングスペースの需要が急増している。」 【설문문】地方都市では、リモートワークの普及によりオフィス需要が減少している。 A. 맞다 / B. 틀리다 / C. 판단할 수 없다

풀이: 본문에서 오피스 수요가 감소한 것은 「도심부」이고, 지방도시에서는 「코워킹 스페이스 수요가 급증」이라고 되어 있습니다. 지방도시의 오피스 수요 감소에 대해서는 언급 없음.

정답: C — 판단할 수 없다

A/B/C 판별 핵심 기준

A (맞다)가 되려면

• 본문의 내용을 그대로 바꿔 말한 것 (패러프레이즈)
• 본문의 두 정보를 조합하면 논리적으로 도출되는 것
• 「AならばB」가 본문에 있고, 설문문도 「AならばB」인 경우

B (틀리다)가 되려면

• 본문과 명백히 반대 — 예: 본문 「증가」 → 설문 「감소」
• 주어/목적어가 뒤바뀐 경우 — 예: 본문 「AがBに影響」→ 설문 「BがAに影響」
• 범위가 다른 경우 — 예: 본문 「一部の企業」→ 설문 「すべての企業」

C (판단불가)가 되려면

• 본문에 해당 주제 자체가 언급되지 않은 경우
• 일반 상식으로는 맞지만 본문에 근거가 없는 경우
• 「かもしれない(일 수도 있다)」 수준으로만 추론 가능한 경우

공략 핵심 — 설문 먼저 읽기 전략

15분 안에 8개 장문(32문)을 풀어야 하므로, 1장문당 약 1분 50초밖에 없습니다. 장문을 처음부터 끝까지 정독할 시간이 없으므로, 설문문을 먼저 읽고 키워드를 파악한 뒤, 본문에서 해당 부분만 찾아 읽는 전략이 유효합니다.

3단계 풀이법

1단계: 설문문 4개를 먼저 스캔 — 키워드 파악 (예: 「地方都市」「オフィス需要」)

2단계: 본문에서 해당 키워드가 등장하는 부분만 집중 독해

3단계: 본문의 서술과 설문문을 대조 — 일치(A), 모순(B), 근거없음(C) 판별

⚠️ 함정 예제 — 상식의 덫

【본문】「AIの進化により、多くの職種で業務効率が向上している。」 【설문문】AIの進化により、一部の職種では雇用が減少している。 → 세상 상식으로는 맞지만, 본문에 「고용 감소」에 대한 언급은 전혀 없습니다.

함정: 「AI → 고용 감소」는 일반적 상식이지만, 이 본문에는 「업무 효율 향상」만 언급. 본문에 없는 내용은 C.

정답: C — 판단할 수 없다 (상식을 섞으면 오답)

시간 배분 가이드

32문 / 15분 = 1문당 약 28초

• 장문 1개 + 설문 4개 = 약 1분 50초 이내
• 설문 스캔: 15초 → 본문 키워드 서치: 40초 → 4문 판별: 55초
• 확신이 없으면 C를 선택하고 넘기는 것도 전략 (오류율 미측정)
• 마지막 2개 장문은 시간이 부족할 수 있으므로, 설문만 보고 직감으로 응답하는 용기도 필요

참고: 아래 예제의 본문은 학습 편의를 위해 짧게 구성했습니다. 실제 시험에서는 500-1000자 정도의 장문이 출제되며, 1개의 장문에 대해 여러 개의 설문이 붙는 구조입니다. 실전 길이 연습은 미니 모의고사에서 할 수 있습니다.

본문을 읽고 각 설문문에 A(맞다) B(틀리다) C(판단불가) 중 하나를 선택하세요.

본문 1 (Q01-Q04)
환경 정책과 기업 활동
近年、環境規制の強化に伴い、多くの製造業企業がCO2排出量の削減に取り組んでいる。特に自動車業界では、電気自動車（EV）の開発が加速しており、2030年までにEV販売比率を50%以上にするという目標を掲げる企業が増えている。一方、EV普及にはバッテリーの原材料であるリチウムの安定供給が課題となっており、リサイクル技術の開発も同時に進められている。なお、CO2削減の手段はEVだけでなく、水素燃料電池や合成燃料など複数の選択肢が検討されている。
Q01
설문
すべての自動車メーカーは、2030年までにEV販売比率を50%以上にする目標を掲げている。
풀이
본문에는 「目標を掲げる企業が増えている」라고 되어 있어 "늘어나고 있다"는 뜻이지 "모든" 기업이라고는 하지 않음. 「すべての(모든)」과 「増えている(늘어나고 있다)」는 범위가 다름.
정답: B — 틀리다 (「모든」이 아니라 「늘어나고 있다」)
Q02
설문
EV普及の課題の一つは、バッテリー原材料の安定供給である。
풀이
본문에 「リチウムの安定供給が課題」라고 직접 언급. 리튬은 배터리 원재료이므로 패러프레이즈.
정답: A — 맞다
Q03
설문
水素燃料電池は、EVよりもCO2削減効果が高い。
풀이
본문에서는 수소연료전지가 CO2 삭감의 선택지 중 하나로 「검토되고 있다」고만 되어 있음. EV와의 효과 비교는 언급 없음.
정답: C — 판단할 수 없다
Q04
설문
CO2排出量の削減に取り組んでいるのは、製造業企業に限られない。
풀이
본문에는 「多くの製造業企業が取り組んでいる」라고 되어 있을 뿐, 제조업에 한정된다/한정되지 않는다에 대한 언급 없음. 일반 상식으로는 제조업 외에도 있겠지만 본문 근거 없음.
정답: C — 판단할 수 없다
본문 2 (Q05-Q08)
일본의 고령화와 노동시장
日本では少子高齢化が進み、生産年齢人口は減少を続けている。この影響で、多くの業界で人手不足が深刻化しており、特に介護·建設·物流の3分野では求人倍率が高い水準にある。これに対応するため、外国人労働者の受け入れ拡大や、定年後の再雇用制度の整備が進んでいる。また、AI·ロボット技術の導入により、単純作業の自動化も加速している。ただし、AIの導入には初期コストが高く、中小企業では普及が遅れているという指摘もある。
Q05
설문
日本の生産年齢人口は、少子高齢化の影響で増加している。
풀이
본문에 「生産年齢人口は減少を続けている」라고 명확히 반대. 증가가 아니라 감소.
정답: B — 틀리다
Q06
설문
介護分野は、人手不足が深刻な分野の一つである。
풀이
본문에 「介護·建設·物流の3分野では求人倍率が高い」라고 직접 언급. 인력부족이 심각한 분야의 하나임이 명확.
정답: A — 맞다
Q07
설문
AI·ロボット技術の導入は、すべての企業規模で順調に進んでいる。
풀이
본문에 「中小企業では普及が遅れているという指摘もある」라고 되어 있어, 모든 기업 규모에서 순조롭게 진행되고 있다는 것과 모순.
정답: B — 틀리다
Q08
설문
外国人労働者の受け入れ拡大により、人手不足は完全に解消された。
풀이
본문에는 외국인 노동자 수용 확대가 「진행되고 있다(進んでいる)」고만 되어 있음. 인력 부족이 「완전히 해소되었다」는 언급 없음. 오히려 여전히 심각하다는 문맥.
정답: B — 틀리다 (「완전히 해소」는 본문과 모순)
본문 3 (Q09-Q10)
소비자 행동의 변화
インターネット通販の拡大により、消費者の購買行動は大きく変化している。実店舗で商品を確認した後にオンラインで購入する「ショールーミング」と呼ばれる行動が一般化し、小売業界に変革を迫っている。これに対して、実店舗では体験型サービスやイベントの開催など、オンラインにはない付加価値の提供に力を入れる動きが広がっている。
Q09
설문
「ショールーミング」とは、オンラインで商品を確認した後に実店舗で購入する行動のことである。
풀이
본문에서 쇼룸밍은 「実店舗で確認 → オンラインで購入」인데, 설문문은 순서가 반대 (「オンラインで確認 → 実店舗で購入」). 주어/목적어 뒤바뀜 함정.
정답: B — 틀리다 (순서가 반대)
Q10
설문
実店舗は、体験型サービスの提供によって、ショールーミングへの対策を行っている。
풀이
본문에 「実店舗では体験型サービスやイベントの開催など、オンラインにはない付加価値の提供に力を入れる」라고 되어 있으며, 이것이 쇼룸밍에 대한 대응이라는 문맥. 패러프레이즈로 A.
정답: A — 맞다
